# 昆明印象 - Kunming Tour

一个展示云南昆明旅游、历史、经济、文化的精美的单页应用网站。

## 项目介绍

"昆明印象"是一个采用现代前端技术构建的昆明城市展示网站。网站采用清新、吸引人的设计风格，通过菜单栏导航的方式，向用户展示昆明这座城市的独特魅力。

### 页面功能

- **首页（这座城）**：介绍昆明的整体印象，展示城市亮点和特色
- **旅游**：展示昆明的著名景点、最佳旅游时间和旅游数据
- **历史**：通过时间线展示昆明两千余年的建城历史和历史古迹
- **经济**：展示昆明的经济数据、支柱产业和发展战略
- **文化**：介绍昆明的民族风情、传统节日、美食文化和艺术传承

## 快速开始

### 环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0 或 pnpm >= 8.0.0

### 安装依赖

```bash
cd kunming-tour
npm install
```

### 开发模式

```bash
npm run dev
```

项目启动后，访问 http://localhost:5173 查看效果。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

## 功能特性

### 🎨 设计特点

- **清新风格**：采用蓝绿渐变、柔和配色，营造春城的清新氛围
- **响应式设计**：完美适配桌面端、平板和手机
- **交互动效**：滚动导航变化、卡片悬停效果、平滑过渡
- **清晰层级**：信息架构层次分明，内容易于阅读

### 📱 响应式布局

- 桌面端（>1024px）：多列网格布局
- 平板端（768px-1024px）：两列布局
- 移动端（<768px）：单列布局，汉堡菜单

### 🧭 导航系统

- 固定顶部导航栏，滚动时自动变化样式
- 桌面端水平菜单，移动端汉堡菜单
- 路由切换自动滚动到顶部
- 页面标题随路由动态更新

## 页面说明

### 首页 - 这座城

- 全屏英雄区，展示昆明城市形象
- 四大亮点卡片：四季如春、高原明珠、民族风情、历史名城
- 城市介绍图文板块
- 快速导航卡片，引导用户深入了解
- 古诗引用，增添文化底蕴

### 旅游页面

- 最佳旅游时间：四季特色介绍
- 必游景点：8个著名景点详细介绍
- 旅游数据概览卡片

### 历史页面

- 时间线展示：从先秦到当代的重要历史事件
- 历史古迹介绍
- 西南联大专题板块

### 经济页面

- 经济数据统计（GDP、人口、人均GDP、财政收入）
- 六大支柱产业介绍
- 城市发展定位进度条
- 区位优势数据展示

### 文化页面

- 标签页切换：民族风情、传统节日、美食文化
- 26个民族的多彩文化展示
- 传统艺术介绍
- 文化数据统计

## 开发指南

### 添加新页面

1. 在 `src/views/` 目录下创建新页面组件
2. 在 `src/router/index.ts` 中添加路由配置
3. 在 `src/components/Navigation.vue` 的 `menuItems` 数组中添加菜单项

### 自定义主题

- Tailwind 配置：通过 `tailwindcss` 插件在 `vite.config.ts` 中配置
- 全局样式：修改 `src/style.css`
- 组件样式：在各 `.vue` 文件的 `<style scoped>` 中定义
