# 气候变化数据仪表盘

一个交互式的气候变化数据可视化仪表盘，展示全球温度、降水量和二氧化碳浓度的变化趋势。

## 功能特性

- 📊 **数据可视化**：支持折线图和柱状图两种展示方式
- 🔄 **动画过渡**：图表切换时带有平滑的动画效果
- 📅 **时间轴选择**：支持近5年、近10年、近20年和全部数据的时间范围筛选
- 🎨 **图例切换**：可点击图例显示/隐藏不同数据系列
- 📥 **数据导出**：支持将数据导出为 Excel 文件
- 🎯 **界面美化**：现代化渐变设计，响应式布局

## 快速开始

### 环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0

### 安装依赖

```bash
cd climate-dashboard
npm install
```

### 开发模式

```bash
npm run dev
```

项目将在 `http://localhost:5173` 启动（如果端口被占用，会自动使用其他端口）。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

## 功能说明

### 1. 数据统计卡片

- **平均温度**：显示当前年份的平均温度及变化趋势
- **年降水量**：显示年降水量统计及年度变化
- **CO₂ 浓度**：显示大气二氧化碳浓度及月度变化

### 2. 时间范围选择

提供两种时间选择方式：
- **快捷按钮**：近 5 年、近 10 年、近 20 年、全部数据
- **滑块控件**：可拖动选择具体年份范围

### 3. 图表类型切换

- **折线图**：适合展示趋势变化
- **柱状图**：适合对比各年份数据

### 4. 图例切换功能

点击图例可以：
- 显示/隐藏对应的数据系列
- 隐藏时数据点变为灰色

### 5. 数据导出

点击"导出数据"按钮，可将当前显示的数据导出为 Excel 文件（.xlsx 格式），包含：
- 年份
- 平均温度 (°C)
- 降水量 (mm)
- CO₂ 浓度 (ppm)

## 数据说明

本项目使用模拟数据用于演示目的，数据生成逻辑包含：
- 温度数据：带有年度上升趋势和季节性波动
- 降水量数据：带有周期性波动和缓慢增长趋势
- CO₂ 浓度数据：持续上升趋势

## 界面设计

### 色彩方案

- **主色调**：蓝色渐变 (#3b82f6 → #2563eb)
- **温度卡片**：红色渐变 (#ef4444 → #f97316)
- **降水卡片**：蓝色渐变 (#3b82f6 → #06b6d4)
- **CO₂ 卡片**：紫色渐变 (#8b5cf6 → #ec4899)

### 动画效果

- **淡入动画**：页面加载时元素从底部淡入
- **滑入动画**：卡片从左右两侧滑入
- **图表动画**：数据更新时的平滑过渡（1500ms）

### 响应式设计

- **移动端**：单列布局，按钮自动换行
- **平板端**：双列/三列自适应
- **桌面端**：完整的多列布局

## 开发说明

### 主要文件说明

1. **`src/main.js`** - 应用入口
   - 配置 PrimeVue 组件（Button、Slider、SelectButton）
   - 配置 Vue Router
   - 导入全局样式

2. **`src/views/DashboardView.vue`** - 主仪表盘
   - 图表渲染（Chart.js）
   - 数据生成逻辑
   - 时间范围筛选
   - 数据导出功能（xlsx）
   - 图例切换逻辑

3. **`src/style.css`** - 全局样式
   - Tailwind CSS 基础样式
   - 自定义动画定义
   - PrimeVue 组件样式覆盖
   - 响应式断点

### 图表配置

所有图表使用 Chart.js 4.x 版本，配置特点：
- 平滑曲线（tension: 0.4）
- 渐变填充
- 悬停放大效果
- 动画缓动函数：easeInOutQuart
