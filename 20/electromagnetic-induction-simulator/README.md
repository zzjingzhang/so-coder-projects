# 电磁感应模拟器

一个交互式的电磁感应原理学习工具，通过可视化动画展示法拉第电磁感应定律和楞次定律。

## 项目简介

本项目是一个基于 Vue 3 的交互式物理教学工具，通过生动的动画演示磁铁与线圈相互作用产生感应电流的过程，帮助学习者理解电磁感应的基本原理。

### 主要功能

- 🧪 **实时电磁感应模拟**：直观展示磁铁移动、磁场变化、感应电流产生的完整过程
- 📊 **动态电流计**：实时显示感应电流的大小和方向
- 🎮 **三种控制模式**：
  - 自动演示：完整的动画演示
  - 手动控制：自由控制磁铁位置
  - 分步演示：逐步骤观察感应过程
- 📝 **知识测验**：包含10道精心设计的题目，覆盖法拉第定律、楞次定律等核心知识点
- 💡 **实时数据监测**：显示感应电流、磁通量变化率、磁铁位置等关键参数

## 技术栈

- **框架**: Vue 3 (Composition API)
- **构建工具**: Vite

## 核心物理原理

### 法拉第电磁感应定律

当穿过闭合回路的磁通量发生变化时，回路中会产生感应电动势。

**数学表达式**：
```
ε = -dΦ/dt
```

其中：
- `ε`：感应电动势（单位：伏特 V）
- `Φ`：磁通量（单位：韦伯 Wb）
- `dΦ/dt`：磁通量随时间的变化率
- 负号：表示感应电动势的方向遵循楞次定律

### 楞次定律

感应电流产生的磁场总是阻碍引起感应电流的磁通量的变化。

**要点说明**：
- 当磁通量增加时，感应电流产生的磁场方向与原磁场方向相反
- 当磁通量减少时，感应电流产生的磁场方向与原磁场方向相同
- 这是能量守恒定律在电磁感应现象中的体现

### 感应电流方向判断

| 场景 | 磁通量变化 | 感应电流方向 | 电流计偏转 |
|------|-----------|-------------|-----------|
| N极接近线圈 | 增加 | 产生反向磁场（阻碍增加） | 负方向 |
| N极离开线圈 | 减少 | 产生同向磁场（阻碍减少） | 正方向 |

## 安装与运行

### 环境要求

- Node.js >= 16.0.0
- npm >= 7.0.0 或 yarn >= 1.22.0

### 安装依赖

```bash
cd electromagnetic-induction-simulator
npm install
```

### 开发模式运行

```bash
npm run dev
```

运行后，在浏览器中访问显示的地址（通常是 `http://localhost:5173`）

### 构建生产版本

```bash
npm run build
```

构建产物将生成在 `dist` 目录中。

### 预览生产版本

```bash
npm run preview
```

## 使用指南

### 模拟实验页面

1. **控制模式选择**：
   - **自动演示**：点击"开始"按钮，磁铁将自动左右移动，完整演示电磁感应过程
   - **手动控制**：通过滑块直接控制磁铁位置，观察不同位置的感应电流
   - **分步演示**：逐步骤观察磁铁移动的关键阶段，配合详细的物理原理说明

2. **动画速度调节**：在自动模式下，可通过滑块调节动画播放速度

3. **实时监测**：右侧面板显示：
   - 感应电流的实时数值
   - 磁通量变化率
   - 磁铁当前位置
   - 电流计指针偏转动画

### 知识测验页面

1. **设置测验**：
   - 选择题目的类别（全部或特定知识点）
   - 选择题目数量（5题或10题）

2. **答题过程**：
   - 每题提供4个选项，选择后点击"提交答案"
   - 提交后显示正确答案和详细解析
   - 可随时返回查看之前的题目

3. **结果分析**：
   - 完成测验后显示得分和正确率
   - 查看每道题的答题情况
   - 支持重新测验或更改设置

## 核心组件说明

### ElectromagneticSimulation.vue

核心动画组件，实现以下功能：

- **磁铁动画**：N极（红色）和S极（蓝色）的可视化
- **磁场线**：从N极出发、回到S极的动态磁场线
- **线圈与感应电流**：12匝线圈的可视化，电流流动动画
- **物理计算**：根据磁铁位置和速度计算感应电流

### Ammeter.vue

电流计组件，特点：

- **半圆刻度盘**：-5A 到 +5A 的量程
- **动态指针**：根据电流大小和方向实时偏转
- **数值显示**：数字显示当前电流值
- **进度条**：直观的水平指示条

## 设计亮点

### 视觉设计

- **深色主题**：采用蓝紫色渐变背景，符合物理实验的科技感
- **清晰的颜色编码**：
  - 红色：N极、负方向电流
  - 蓝色：S极
  - 青色：正方向电流
  - 靛蓝色：主题色

### 用户体验

- **响应式设计**：适配不同屏幕尺寸
- **流畅动画**：CSS Animation + requestAnimationFrame 实现高性能动画
- **直观交互**：按钮文字水平垂直居中，hover 效果明显
- **分步引导**：分步模式提供详细的物理原理说明
