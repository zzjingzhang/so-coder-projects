# 立方体观察工具 (Cube Visualizer)

一个用于小学四年级数学教学的互动立方体观察工具，帮助学生直观理解立方体在不同角度的样子。

## 项目介绍

小刚老师专用的立方体观察工具，用于教授空间立体图形知识，特别是立方体的不同视图和观察角度。

### 主要功能

- **3D 立体视图**：展示 4 个立方体组成的立体结构，其中一个立方体可以改变位置
- **自由旋转**：通过鼠标拖动自由旋转立体结构，从不同角度观察
- **缩放功能**：支持缩放操作，方便细节观察
- **截图功能**：一键保存当前视图为图片，用于课件和习题
- **正交视图**：实时展示正视图、左视图和俯视图
- **面标签**：每个立方体的六个面都有颜色和文字标记（前、后、左、右、上、下）
- **位置调整**：可以通过按钮移动其中一个立方体的位置

## 技术栈

- **React**：用于构建用户界面的 JavaScript 库
- **Vite**：下一代前端构建工具
- **Tailwind CSS**：实用优先的 CSS 框架
- **Ant Design**：企业级 UI 设计语言和 React 组件库
- **React Router**：React 应用的声明式路由
- **html2canvas**：用于将 DOM 元素转换为 Canvas 图片

## 安装和运行

### 前置要求

确保你的开发环境已安装：
- Node.js (推荐版本 16 或更高)
- npm (通常随 Node.js 一起安装)

### 安装依赖

```bash
cd cube-visualizer
npm install
```

### 启动开发服务器

```bash
npm run dev
```

开发服务器启动后，你可以在浏览器中访问显示的本地地址（通常是 `http://localhost:5173` 或类似地址）来查看应用。

### 构建生产版本

```bash
npm run build
```

构建完成后，生产版本的文件将生成在 `dist` 目录中。

### 预览生产版本

```bash
npm run preview
```

## 使用说明

### 基本操作

1. **旋转视图**：在 3D 视图区域按住鼠标左键并拖动，可以自由旋转立体结构
2. **调整角度**：使用下方的滑块可以精确调整 X 轴和 Y 轴的旋转角度
3. **缩放**：使用缩放滑块或缩放按钮调整视图大小
4. **截图**：点击右上角的"截图"按钮，当前视图将保存为图片文件
5. **重置**：点击"重置"按钮将所有设置恢复到默认状态

### 立方体位置调整

在右侧控制面板中，你可以调整可移动立方体的位置：

- **上下移动**：点击上下箭头按钮
- **左右移动**：点击左右箭头按钮
- **前后移动**：点击前后按钮

注意：可移动立方体不能与固定立方体重叠。

### 快速视角切换

点击右侧的快速视角按钮可以快速切换到标准视图：

- **正视图**：从正前方观察
- **左视图**：从左侧观察
- **俯视图**：从正上方观察
- **默认视角**：回到初始的 3D 视角

### 面标签和颜色

每个立方体的六个面都有独特的颜色和文字标记：

- **前面**：红色
- **后面**：橙色
- **右面**：绿色
- **左面**：蓝色
- **上面**：黄色
- **下面**：紫色

你可以通过"显示面标签"/"隐藏面标签"按钮切换文字标记的显示。

### 正交视图

页面底部展示了三个正交视图：

- **正视图**：从正前方看到的形状
- **左视图**：从左侧看到的形状
- **俯视图**：从正上方看到的形状

这些视图会根据立方体的位置实时更新，帮助学生理解不同观察角度下的图形变化。

## 教学建议

1. **基础概念讲解**：使用颜色和标签帮助学生建立前后左右上下的空间概念
2. **视角对比**：切换不同视角，让学生观察同一个立体结构在不同角度的样子
3. **互动练习**：移动可移动立方体，让学生预测正交视图的变化
4. **截图保存**：将重要的教学视角保存为图片，用于制作课件或习题
5. **实际物体对比**：结合实际的立方体教具，帮助学生建立从虚拟到现实的联系

## 开发说明

### 项目配置

- **Tailwind CSS**：已配置并启用，支持响应式设计
- **Ant Design**：已集成，用于 UI 组件
- **React Router**：已配置，支持单页应用路由
- **html2canvas**：已集成，用于截图功能

### 组件说明

- **CubeVisualizer**：主组件，包含所有功能逻辑
- **Cube**：立方体组件，渲染单个立方体及其六个面
- **OrthographicView**：正交视图组件，显示正视图、左视图或俯视图

### 自定义开发

如果你想扩展或修改这个工具：

1. **修改立方体数量**：在 `CubeVisualizer.jsx` 中修改 `fixedCubes` 数组
2. **调整颜色**：在 `CubeVisualizer.css` 中修改各面的颜色
3. **添加新功能**：在 `CubeVisualizer.jsx` 中添加新的状态和函数
