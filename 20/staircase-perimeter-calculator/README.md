# 楼梯图形平移法计算周长

一个动态的教育网页，展示"楼梯"图形平移法计算周长的数学原理。

## 项目简介

本项目通过精美的动画效果，直观地展示了如何使用平移法计算楼梯形状的周长。通过将水平边和垂直边分别平移，我们可以将复杂的楼梯图形转化为简单的长方形，从而使用长方形周长公式进行计算。

## 动画流程

1. **标题淡入**: 页面加载后，标题带有发光效果淡入
2. **显示楼梯**: 展示原始的楼梯图形结构
3. **显示边界**: 显示目标长方形的虚线边界
4. **水平平移**: 水平边分阶段向右平移
5. **垂直平移**: 垂直边分阶段向下平移
6. **合并图形**: 楼梯图形变成长方形
7. **显示公式**: 展示长方形周长计算公式 (3+4)×2=14

## 数学原理

楼梯图形的周长计算可以通过平移法简化：

- **水平边总和**: 所有水平边的长度之和等于长方形的长 × 2
- **垂直边总和**: 所有垂直边的长度之和等于长方形的宽 × 2
- **周长公式**: 周长 = (长 + 宽) × 2

在本示例中：
- 长 = 4
- 宽 = 3
- 周长 = (4 + 3) × 2 = 14
