# 虚拟生态系统瓶 (Virtual Ecosystem Bottle)

一个交互式的虚拟水族生态系统模拟器，展示植物、动物和微生物之间的氧气与二氧化碳平衡。

## 项目简介

本项目模拟了一个封闭的水族生态系统，用户可以通过添加不同类型的生物（水草、藻类、小鱼、虾、细菌和原生生物），观察它们之间的气体交换（光合作用和呼吸作用），以及生态系统的动态平衡。

## 功能特性

### 🐠 生态系统模拟
- **生物多样性**：支持添加6种不同类型的生物
  - 植物类：水草、藻类
  - 动物类：小鱼、虾
  - 微生物类：细菌、原生生物
- **气体交换**：真实模拟光合作用和呼吸作用
  - 植物：吸收CO₂，释放O₂
  - 动物/微生物：消耗O₂，产生CO₂
- **动态平衡**：实时计算氧气和二氧化碳浓度变化

### 🎮 交互控制
- **系统控制**：启动、暂停、重置模拟
- **速度调节**：0.5x - 3x 模拟速度
- **环境参数**：
  - 光照强度（0-100%）
  - 温度（10-35°C）
  - 营养水平（0-100%）
- **生物添加**：选择生物类型后点击水族箱添加

### 📊 数据可视化
- **实时监测**：
  - 氧气浓度 (%)
  - 二氧化碳浓度 (%)
  - pH值
  - 系统运行时间
- **统计数据**：
  - 植物/动物/微生物总量
  - 光合作用率
  - 呼吸作用率
  - 生态健康度
- **趋势图表**：气体浓度变化曲线图 (Chart.js)

### 🎨 视觉效果
- **玻璃态设计**：半透明毛玻璃效果
- **生物动画**：
  - 鱼类：左右游动
  - 虾：跳跃动画
  - 植物：随风摇摆
  - 微生物：漂浮移动
- **气泡效果**：植物光合作用时产生上升气泡
- **响应式布局**：适配各种屏幕尺寸

### 📚 科学知识
- 光合作用原理与化学方程式
- 呼吸作用原理与化学方程式
- 气体平衡机制
- 微生物在生态系统中的作用
- 环境因素（光照、温度、营养）的影响
- pH值与水质关系

## 快速开始

### 环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0 或 yarn >= 1.22.0

### 安装依赖

```bash
cd virtual-ecosystem-bottle
npm install
```

### 开发模式

```bash
npm run dev
```

项目将在 `http://localhost:5173` 启动。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

## 使用说明

### 1. 添加生物
1. 在左侧控制面板的「添加生物」区域选择生物类型
2. 点击右侧水族箱的任意位置添加生物
3. 可添加多种生物观察它们的交互

### 2. 控制模拟
- **启动**：开始生态系统模拟
- **暂停**：暂停模拟但保留当前状态
- **重置**：重置所有状态到初始值
- **速度**：调节模拟速度（0.5x - 3x）

### 3. 调节环境
- **光照强度**：影响光合作用效率
- **温度**：影响所有生物的代谢速率
- **营养水平**：影响植物生长和微生物活性

### 4. 观察数据
- 查看底部数据面板的实时监测数据
- 观察气体浓度变化趋势图
- 关注生态健康度评分

## 生态系统原理

### 光合作用 (Photosynthesis)
植物和藻类利用光能将二氧化碳和水转化为葡萄糖（能量）和氧气。

```
6CO₂ + 6H₂O + 光能 → C₆H₁₂O₆ + 6O₂
```

### 呼吸作用 (Respiration)
所有生物（包括植物）分解有机物释放能量，消耗氧气产生二氧化碳。

```
C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + 能量
```

### 生态平衡
健康的生态系统中：
- 植物产生的氧气 ≈ 所有生物消耗的氧气
- 动物产生的二氧化碳 ≈ 植物消耗的二氧化碳
- pH值维持在 6.5-7.5 之间

## 开发说明

### 组件通信
- 使用 Vue 3 的 `defineProps` 和 `defineEmits` 进行父子组件通信
- 主逻辑集中在 `HomeView.vue`，状态向下传递到各子组件

### 动画实现
- 使用 CSS `@keyframes` 定义基础动画
- 每个生物类型有独特的动画类
- 气泡使用动态创建和定时清理

### 数据更新
- 使用 `setInterval` 每 100ms 更新一次生态系统状态
- 图表数据每 2 秒记录一次历史点
- 使用 Vue 3 的响应式系统自动更新 UI
