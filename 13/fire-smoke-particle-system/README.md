# 火灾与烟雾粒子系统 (Fire & Smoke Particle System)

一个使用 Vue 3 + TypeScript 构建的真实感火灾和烟雾粒子系统模拟，具有实时风相互作用效果。

## 功能特性

- 🔥 **真实火灾模拟**：包含火焰、烟雾和火花三种粒子类型
- 💨 **风相互作用**：实时调节风速观察风对火焰的影响
- 🌪️ **湍流效果**：使用 Perlin 噪声实现自然的湍流运动
- 🎮 **实时控制**：通过滑块实时调节粒子系统参数
- 🖱️ **交互功能**：点击画布可在任意位置添加额外火焰点

## 可调节参数

| 参数 | 说明 | 范围 |
|------|------|------|
| 发射率 | 每帧发射的粒子数量 | 1 - 20 |
| 粒子大小 | 粒子的基础尺寸 | 5px - 50px |
| 湍流强度 | 粒子运动的不规则程度 | 0 - 5 |
| 风速 | 风对粒子的水平影响 | -10 (左) - 10 (右) |
| 火焰强度 | 火焰的燃烧强度和上升速度 | 0.1x - 2x |
| 烟雾密度 | 烟雾的浓密程度 | 0% - 100% |

## 技术栈

- **框架**: Vue 3 (Composition API)
- **语言**: TypeScript
- **样式**: Tailwind CSS v4
- **UI 组件库**: Ant Design Vue
- **路由**: Vue Router
- **打包工具**: Vite

## 安装与运行

### 前置要求

- Node.js >= 16
- npm 或 pnpm 或 yarn

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

开发服务器将在 `http://localhost:5173` 启动（端口可能因环境而异）。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist/` 目录。

### 预览生产版本

```bash
npm run preview
```

## 使用说明

1. **启动项目**：运行 `npm run dev` 并在浏览器中打开
2. **观察效果**：默认在画布底部中央有一个持续燃烧的火焰
3. **调节参数**：使用右侧控制面板的滑块实时调节粒子系统
4. **交互操作**：
   - 点击画布任意位置可在该位置添加额外火焰和火花
   - 调节风速观察火焰被风吹动的效果
   - 增加湍流强度使火焰更加不规则和自然

## 粒子系统实现细节

### 粒子类型

1. **火焰粒子**：
   - 颜色从亮黄色逐渐过渡到橙色再到红色
   - 具有向上的浮力加速度
   - 粒子大小随生命逐渐缩小

2. **烟雾粒子**：
   - 灰色半透明效果
   - 上升速度较慢
   - 粒子大小随生命逐渐扩大
   - 透明度随生命逐渐降低

3. **火花粒子**：
   - 小而亮的火星效果
   - 速度较快，轨迹更随机
   - 具有闪烁效果

### 物理模拟

- **风场**：恒定水平力影响所有粒子
- **湍流**：使用 Perlin 噪声生成随时间变化的随机力场
- **浮力**：火焰和烟雾粒子受到向上的恒定加速度
- **重力**：火花粒子受到向下的重力加速度
