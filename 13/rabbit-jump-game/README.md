# 🐰 兔子跳台阶 - 趣味教学小游戏

一个帮助小学生理解组合数学问题的互动教学游戏。通过可爱的兔子跳台阶动画，让孩子们直观地理解"斐波那契数列"和"组合路径"的数学概念。

## 📋 项目简介

这是一个专为小学数学教学设计的互动游戏。游戏展示了一个经典的组合数学问题：
- 一只兔子可以一次跳 1 级或 2 级台阶
- 它跳完 n 级台阶有多少种不同的跳法？

通过实际操作，让学生亲自探索不同的跳跃路径，理解递推和组合的数学原理。

## ✨ 游戏功能

### 核心功能
- **台阶数量设置**：可选择 1-10 级台阶
- **兔子跳跃动画**：点击按钮让兔子跳 1 级或 2 级台阶
- **跳跃轨迹记录**：实时显示当前的跳跃步骤
- **跳法收集**：自动记录所有找到的不同跳法
- **撤销功能**：可以撤销上一步跳跃
- **数学提示**：展示斐波那契数列的数学原理

- 🎨 可爱的渐变背景

## 🚀 快速开始

### 前置要求
确保你的电脑已经安装了以下软件：
- Node.js (建议版本 18.x 或更高)
- npm (通常随 Node.js 一起安装)

### 安装步骤

1. **进入项目目录**
```bash
cd rabbit-jump-game
```

2. **安装依赖**
```bash
npm install
```

3. **启动开发服务器**
```bash
npm run dev
```

4. **访问应用**
打开浏览器，访问显示的本地地址（通常是 `http://localhost:5173` 或附近的端口）

### 其他命令

- **构建生产版本**
```bash
npm run build
```

- **预览生产版本**
```bash
npm run preview
```

## 🎮 游戏玩法说明

1. **设置台阶数量**：使用 +/- 按钮调整台阶数量（1-10 级）
2. **开始游戏**：点击"开始游戏"按钮
3. **让兔子跳跃**：
   - 点击"跳 1 级"让兔子跳上 1 级台阶
   - 点击"跳 2 级"让兔子跳上 2 级台阶
4. **查看状态**：左侧面板显示当前位置、跳跃轨迹和已找到的跳法
5. **撤销操作**：如果跳错了，可以点击"撤销上一步"
6. **完成挑战**：当兔子到达终点时，会记录这次的跳法
7. **探索更多**：点击"再试一次"，尝试找出所有不同的跳法！

## 🧮 数学原理

这个游戏展示了著名的**斐波那契数列**问题：

- 跳上 1 级台阶：只有 1 种方法（跳 1 级）
- 跳上 2 级台阶：有 2 种方法（1+1 或直接跳 2）
- 跳上 n 级台阶：方法数 = 跳上 (n-1) 级的方法数 + 跳上 (n-2) 级的方法数

**为什么？**
因为到达第 n 级台阶的最后一步，要么是从第 n-1 级跳 1 级上来，要么是从第 n-2 级跳 2 级上来。
