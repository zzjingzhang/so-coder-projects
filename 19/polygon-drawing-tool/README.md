# 多边形绘制工具

一个基于 Vue 3 的多边形绘制工具，支持 GeoGebra 格式的多边形数据解析和可视化。

## 项目结构

```
polygon-drawing-tool/
├── public/
│   ├── favicon.svg
│   └── icons.svg
├── src/
│   ├── components/
│   │   ├── CanvasDisplay.vue    # 画布显示组件（坐标系和图形绘制）
│   │   └── DataInput.vue        # 数据输入组件
│   ├── router/
│   │   └── index.js             # 路由配置
│   ├── views/
│   │   └── PolygonDrawing.vue   # 主页面视图
│   ├── App.vue                  # 根组件
│   ├── main.js                  # 入口文件
│   └── style.css                # 全局样式
├── index.html                   # HTML 入口
├── package.json                 # 项目配置
├── postcss.config.js            # PostCSS 配置
├── tailwind.config.js           # Tailwind CSS 配置
└── vite.config.js               # Vite 配置
```

## 技术栈

- **框架**: Vue 3 (Composition API)
- **语言**: JavaScript
- **样式**: Tailwind CSS
- **UI 组件库**: Ant Design Vue
- **路由**: Vue Router 4
- **打包工具**: Vite

## 功能特性

- ✅ 页面初始化时右侧显示坐标系
- ✅ 左侧数据输入区支持 GeoGebra 多边形格式
- ✅ 多个多边形使用换行符间隔
- ✅ 点击绘制按钮在右侧坐标系中显示图形
- ✅ 自动缩放和平移以适应所有多边形
- ✅ 支持多个多边形同时显示，不同颜色区分
- ✅ 实时响应窗口大小变化
- ✅ 图例显示，便于识别不同多边形
- ✅ 操作反馈（成功/错误消息提示）

## 数据格式

支持 GeoGebra 多边形数据格式：

```
Polygon[(x1,y1), (x2,y2), (x3,y3), ...]
```

### 示例

单个多边形：
```
Polygon[(-13851.731323130303, -24424.111943115407), (-14056.006243860069, -24863.340584631682), (-14390.735828431802, -24691.257842082967), (-14385.792668078786, -24426.083320980273), (-14143.075891070905, -24297.028521422708), (-13995.667338693806, -24043.61919395542), (-13764.307490452862, -24076.185910189484), (-13667.569411966626, -24258.1237746053), (-13851.731323130303, -24424.111943115407)]
```

多个多边形（换行分隔）：
```
Polygon[(0,0), (100,0), (100,100), (0,100)]
Polygon[(50,50), (150,50), (150,150), (50,150)]
Polygon[(25,75), (75,25), (125,75), (75,125)]
```

## 安装和运行

### 环境要求

- Node.js >= 14.0.0
- npm >= 6.0.0

### 安装依赖

```bash
cd polygon-drawing-tool
npm install
```

### 开发模式

```bash
npm run dev
```

项目将在 `http://localhost:5173` 启动。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

## 使用说明

1. 打开应用后，左侧为数据输入区，右侧为坐标系视图
2. 在左侧文本框中输入 GeoGebra 格式的多边形数据
3. 每个多边形占一行，使用 `Polygon[(x1,y1), (x2,y2), ...]` 格式
4. 点击「绘制图形」按钮，右侧坐标系将显示解析后的多边形
5. 点击「清空」按钮可以清空输入数据和已绘制的图形
6. 坐标系会自动缩放以适应所有多边形，并居中显示

## 组件说明

### DataInput.vue (数据输入组件)

- 提供文本区域用于输入多边形数据
- 支持解析 GeoGebra 多边形格式
- 包含绘制和清空按钮
- 提供数据格式说明和使用指南
- 操作反馈（消息提示）

### CanvasDisplay.vue (画布显示组件)

- 初始化时显示笛卡尔坐标系（带网格线）
- 接收多边形数据并在画布上绘制
- 自动计算缩放比例和偏移量以适应所有点
- 支持多个多边形同时显示（不同颜色）
- 显示图例，便于识别不同多边形
- 响应窗口大小变化

### PolygonDrawing.vue (主页面视图)

- 布局管理：左侧输入区（1/3宽度），右侧画布区（2/3宽度）
- 组件组合：DataInput + CanvasDisplay
- 数据传递：通过 props 和事件实现组件间通信

## 设计特点

- **色彩搭配**：使用柔和的灰色调背景，蓝色为主色调，避免视觉疲劳
- **响应式设计**：适配不同屏幕尺寸，画布自动调整大小
- **用户体验**：
  - 按钮文字水平垂直居中
  - 操作有明确的反馈（成功/错误提示）
  - 输入区域有占位符提示数据格式
  - 图例清晰标识不同多边形
- **代码组织**：
  - 组件职责单一
  - 使用 Composition API 提高代码可维护性
  - 样式使用 Tailwind CSS 实现快速开发

## 注意事项

1. 多边形数据必须至少包含 3 个顶点才能被解析
2. 坐标值可以是整数或浮点数，支持正负值
3. 多个多边形之间必须使用换行符分隔
4. 坐标系会自动缩放以适应所有多边形，可能会改变原始比例
5. Y 轴在画布上是向下增长的，绘制时会自动翻转以符合数学坐标系习惯
