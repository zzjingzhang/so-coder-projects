# 📻 摩尔斯电码练习工具

一个专为孩子们设计的摩尔斯电码学习工具，让学习变得有趣又简单！

## ✨ 功能特性

### 🎮 两种练习模式
- **简单模式**：通过按钮轻松输入点(.)和划(-)，适合初学者快速入门
- **电报机模式**：模拟真实电报机操作，体验历史上真实的无线电通信方式

### 📋 完整功能
- **摩尔斯电码对照表**：查看所有字母(A-Z)和数字(0-9)对应的电码
- **即时解码**：输入的电码即时显示对应的字母解析
- **语音播放**：支持播放摩尔斯电码的声音信号
- **文字转电码**：输入文字自动转换为摩尔斯电码
- **自定义设置**：调整按键时间和间隔时间，适应不同的学习速度
- **暗色模式**：支持亮/暗主题切换，保护眼睛

### ⚙️ 可配置选项
- 点持续时间
- 划持续时间  
- 元素间隔时间
- 字母间隔时间
- 单词间隔时间

## 🛠️ 技术栈

- **框架**: React 19
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **UI 组件库**: Ant Design
- **路由**: React Router v6
- **打包工具**: Vite 8

## 🚀 快速开始

### 环境要求
- Node.js >= 18.0.0
- npm >= 9.0.0

### 安装依赖

```bash
cd morse-code-practice
npm install
```

### 启动开发服务器

```bash
npm run dev
```

启动后，浏览器会自动打开 `http://localhost:5173`（或其他可用端口）

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 📖 使用说明

### 简单模式
1. 点击「点」按钮输入点符号 (.)
2. 点击「划」按钮输入划符号 (-)
3. 点击「字母间隔」分隔字母
4. 点击「单词间隔」分隔单词
5. 点击「播放」按钮收听声音
6. 输入文本可以自动转换为摩尔斯电码

### 电报机模式
1. **短按**电报键 = 点 (.)
2. **长按**电报键 = 划 (-)
3. 松开后会自动等待字母间隔
4. 点击「单词间隔」按钮分隔单词
5. 进度条可以帮助判断按下的时间

### 设置
1. 调整各种时间参数以适应学习进度
2. 初学者建议使用较长的间隔时间
3. 进阶者可以缩短时间提高难度
4. 点击「恢复默认设置」重置所有参数

## 🎯 摩尔斯电码规则

### 基本信号
- **点 (Dot ·)**: 1 个时间单位
- **划 (Dash —)**: 3 个时间单位

### 间隔规则
- 点/划之间间隔: 1 个时间单位
- 字母之间间隔: 3 个时间单位  
- 单词之间间隔: 7 个时间单位

### 默认时间设置
- 点: 200ms
- 划: 600ms (3倍点)
- 元素间隔: 200ms
- 字母间隔: 600ms (3倍点)
- 单词间隔: 1400ms (7倍点)

## 🔧 开发说明

### 目录说明
- `components/`: 所有 React 组件
- `types/`: TypeScript 类型定义
- `utils/`: 工具函数（摩尔斯电码转换逻辑）

### 核心组件
- `App.tsx`: 主应用，包含路由配置、状态管理、导航
- `MorseCodeTable.tsx`: 电码对照表展示
- `SimpleMode.tsx`: 简单模式实现
- `TelegraphMode.tsx`: 电报机模式实现
- `Settings.tsx`: 设置界面

### 工具函数
- `encodeToMorse()`: 文字转摩尔斯电码
- `decodeFromMorse()`: 摩尔斯电码转文字
- `isDot()` / `isDash()`: 判断按下时间是点还是划

## 📝 注意事项

1. **音频权限**: 首次使用播放功能时，浏览器可能会请求音频权限，请允许
2. **移动端适配**: 应用已适配移动端，可以在手机和平板上使用
3. **数据存储**: 设置会自动保存到浏览器本地存储（localStorage）
4. **暗色模式**: 可以通过右上角的开关切换亮/暗主题
