# 🎨 涂鸦防线

一款独特的塔防小游戏，玩家通过在屏幕上绘制不同颜色和形状的线条来引导、阻碍并消灭敌人。采用手绘涂鸦风格，色彩鲜明，充满童趣。

## 🎮 游戏玩法

### 核心机制
- **线条即陷阱**：不同颜色的画笔拥有不同效果
- **有限资源**：每一关提供有限的"颜料"作为资源
- **策略解谜**：观察敌人移动模式，设计最优防御路线

### 画笔效果
| 颜色 | 名称 | 效果 | 颜料消耗 |
|------|------|------|----------|
| 🔴 红色 | 火焰线条 | 灼烧敌人，造成持续伤害。敌人离开后仍会燃烧一段时间 | 0.15/像素 |
| 🔵 蓝色 | 黏液线条 | 大幅减缓敌人移动速度。让敌人在其他陷阱中停留更久 | 0.10/像素 |
| 🟡 黄色 | 磁力线条 | 吸引附近的敌人向线条靠近。可以用来聚集敌人 | 0.20/像素 |
| ⚫ 黑色 | 墙壁线条 | 完全阻挡敌人，迫使其绕道而行。有耐久度限制 | 0.25/像素 |

### 敌人类型
| 类型 | 特点 | 生命值 | 速度 |
|------|------|--------|------|
| 😈 普通涂鸦 | 标准敌人，没有特殊能力 | 100 | 1.5 |
| 💨 快速涂鸦 | 移动速度快，但生命值较低 | 60 | 3.0 |
| 🛡️ 重装涂鸦 | 生命值高，但移动缓慢 | 300 | 0.8 |
| 🔥 抗火涂鸦 | 对火焰伤害免疫，但更容易被减速 | 120 | 1.2 |
| ❄️ 抗冰涂鸦 | 对减速效果免疫，但更容易被火烧 | 120 | 1.2 |

## 🎯 关卡设计

游戏包含5个精心设计的关卡：

1. **🌱 初学者画廊** - 学习基本的涂鸦防御技巧
2. **🔬 色彩实验室** - 尝试不同颜色的组合效果
3. **⚔️ 涂鸦竞技场** - 面对更多样化的敌人
4. **🌊 元素风暴** - 对抗具有元素抗性的敌人
5. **🏆 终极挑战** - 所有技能的终极考验

## 🎨 设计风格

### 手绘涂鸦风
- 所有游戏元素采用铅笔素描和彩色蜡笔涂鸦风格
- 线条粗犷，色彩鲜明，略带童趣
- 画布背景模拟纸张纹理

### 动态效果
- 绘制线条时有真实的笔触抖动效果
- 火焰线条有闪烁的火苗动画
- 黏液线条有滴落效果
- 敌人被消灭时化为彩色粒子

## 🧠 核心算法

### A* 寻路算法
敌人使用A*算法自动寻找最短路径，墙壁线条会动态改变寻路结果。

### 碰撞检测
- 线段与圆形的精确碰撞检测
- 支持不同厚度的线条
- 实时计算敌人与线条的距离

### 游戏循环
- 使用 `requestAnimationFrame` 实现60fps游戏循环
- 基于时间的状态更新，确保不同设备上体验一致
- 粒子系统实现视觉特效

## 📝 开发说明

### 路径别名
项目配置了 `@` 别名指向 `src` 目录，可以使用如下导入方式：

```typescript
import { GameCanvas } from '@/components/GameCanvas';
import { useGameLogic } from '@/hooks/useGameLogic';
import { LEVELS } from '@/types';
```

### 添加新关卡
在 `src/types/index.ts` 的 `LEVELS` 数组中添加新关卡配置：

```typescript
{
  id: 6,
  name: '新关卡名称',
  description: '关卡描述',
  startPoint: { x: 50, y: 300 },
  endPoint: { x: 750, y: 300 },
  enemyWaves: [
    {
      delay: 2000,
      enemies: [
        { type: 'normal', count: 5, interval: 1500 },
      ],
    },
  ],
  initialPaint: 500,
  obstacles: [],
}
```

## 🎮 操作指南

1. **选择关卡**：在首页点击关卡卡片开始游戏
2. **选择画笔**：在左侧工具栏点击不同颜色的画笔
3. **绘制线条**：在画布上按住鼠标拖动绘制线条
4. **观察效果**：敌人会自动寻路并受到线条效果影响
5. **完成关卡**：消灭所有敌人即可胜利
