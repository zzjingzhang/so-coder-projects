# SafeHaven - 儿童青少年心理支持平台

🌻 **SafeHaven** 是一个专门为感到社会孤立或在学校遭受欺凌的儿童和青少年提供支持的平台。该平台提供工具和资源来帮助他们应对挑战、建立信心和发展社交技能。

## 🎯 项目简介

### 设计目标
- **温暖、诱人且适合儿童** - 采用友好的颜色和界面设计
- **强调安全、信任和授权** - 为用户提供一个安全的交流空间
- **体现专业人士和同行共同努力的协作氛围** - 结合专家指导和同伴支持

### 目标受众
- **8-16岁儿童和青少年** - 主要用户群体
- **父母或监护人** - 提供家长资源和支持

### 设计基调
- **支持性** - 温暖、关怀的用户体验
- **授权性** - 帮助用户建立自信
- **富有同情心** - 理解和尊重用户感受
- **专业性** - 基于专业心理学知识

## 📄 页面功能介绍

### 1. 首页 (`/`)
- 平台价值主张介绍
- 核心功能展示
- 用户统计数据
- 用户评价轮播
- 行动呼吁区域

### 2. 专业指导页面 (`/professionals`)
- 多种咨询方式介绍（视频咨询、实时聊天、策划内容）
- 专家团队展示（可按在线状态筛选）
- 专家详情卡片
- 在线聊天和视频预约弹窗
- 自助资源库
- 紧急求助入口

### 3. 自助工具页面 (`/tools`)
- 情绪追踪器（可记录每日心情和备注）
- 自信培养游戏（每日小挑战）
- 情绪调节练习（4-7-8呼吸法、5-4-3-2-1感官接地等）
- 更多工具分类展示
- 成长仪表盘示例
- 安全和隐私说明

### 4. 社区交流页面 (`/community`)
- 安全公告栏
- 兴趣群组分类筛选
- 群组卡片展示（含成员数、在线人数）
- 近期活动列表
- 社区安全指南（隐私保护、友善相处、拒绝欺凌等）
- 加入和创建群组引导

### 5. 家长资源页面 (`/parents`)
- 需要关注的警示信号
- 有效沟通技巧
- 专家文章推荐
- 常见问题解答（FAQ）
- 紧急求助热线展示
- 家长账号注册引导

### 6. 登录/注册页面 (`/login`)
- 登录和注册选项卡切换
- 用户类型选择（青少年/家长）
- 完整的表单验证
- 社交媒体登录选项
- 安全和隐私提示
- 返回首页链接

## 🎨 设计特点

### 颜色方案
- **主色调 (Primary)**：绿色系 (#22c55e) - 代表希望和成长
- **强调色 (Accent)**：蓝色系 (#0ea5e9) - 代表专业和信任
- **辅助色 (Secondary)**：橙色系 (#d97706) - 代表温暖和活力

### 组件样式
- `btn-primary` - 主按钮样式
- `btn-secondary` - 次按钮样式
- `card` - 卡片组件样式
- `nav-link` - 导航链接样式
- `gradient-text` - 渐变文字效果
- `animate-float` - 浮动动画效果

## 🚀 快速开始

### 环境要求
- Node.js >= 16.0.0
- npm >= 7.0.0

### 安装依赖

```bash
# 进入项目目录
cd safe-haven

# 安装依赖
npm install
```

### 开发模式

```bash
# 启动开发服务器
npm run dev
```

开发服务器默认运行在 `http://localhost:5173`

### 构建生产版本

```bash
# 构建生产版本
npm run build
```

构建产物将输出到 `dist` 目录

### 预览生产版本

```bash
# 预览生产构建
npm run preview
```
