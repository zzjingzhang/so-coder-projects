# 1-25 数学消消乐游戏

一个使用 Angular 框架开发的1-25数字消消乐小游戏。

## 游戏规则

- 游戏开始后，屏幕上会显示 1-25 随机排列的数字按钮
- 玩家需要按照 **1 → 2 → 3 → ... → 25** 的顺序依次点击数字
- 点击正确的数字后，该数字会消失
- 如果 **点错数字**，所有已消除的数字会 **全部重新出现**，需要从 1 开始重新点击
- 全部消除后，会显示完成游戏所用的时间

## 核心功能

1. **随机排列**: 每次游戏开始时，1-25 的数字会随机排列在 5×5 的网格中
2. **顺序验证**: 严格按照 1→2→3...→25 的顺序验证点击
3. **错误重置**: 点错后所有已消除的数字会重新出现
4. **实时计时**: 显示游戏进行时间，精确到百分之一秒
5. **进度显示**: 实时显示已消除数字数量 / 总数量
6. **完成弹窗**: 全部消除后弹出恭喜窗口，显示总用时
7. **重新开始**: 随时可以重置游戏

## 快速开始

### 环境要求

- Node.js 18.x 或更高版本
- npm (随 Node.js 一起安装)
- Angular CLI (可选，npm scripts 已配置)

### 安装依赖

```bash
cd number-match-game
npm install
```

### 启动开发服务器

```bash
npm start
# 或者
ng serve
```

启动后，在浏览器中打开 `http://localhost:4200/` 即可开始游戏。

### 构建生产版本

```bash
npm run build
# 或者
ng build
```

构建产物将输出到 `dist/` 目录。

### 运行测试

```bash
npm test
# 或者
ng test
```

## 使用说明

1. 打开页面后，游戏会自动开始
2. 找到数字 **1** 并点击，点击正确后数字会消失
3. 继续点击 **2**、**3**... 以此类推
4. 如果不小心点错了数字，所有已消失的数字会重新出现，需要从 1 开始重新点击
5. 全部消除后，会弹出窗口显示完成时间
6. 点击"再玩一次"或"重新开始"按钮可以开始新一局游戏

## 技术亮点

1. **Angular Standalone Components**: 使用最新的独立组件模式，无需 NgModule
2. **纯 Tailwind CSS 实现**: 所有 UI 组件完全使用 Tailwind CSS 自定义实现，无第三方 UI 库依赖
3. **响应式设计**: 完美适配各种屏幕尺寸（移动端、平板、桌面端）
4. **流畅动画**: 正确点击脉冲动画、错误点击抖动动画、数字消失过渡效果
5. **渐变色彩**: 使用现代化的渐变色彩方案，视觉效果出色
6. **性能优化**: 使用高效的状态管理和变更检测
7. **轻量级**: 移除了不必要的依赖，项目更加精简
