# 🐍 贪吃蛇游戏 (Snake Game)

一个现代化、富有科技感的贪吃蛇游戏应用，支持多设备控制、响应式设计和特色障碍系统。

## ✨ 功能特性

### 🎮 核心玩法
- **经典贪吃蛇**：控制蛇吃食物来增长身体和获得分数
- **多设备控制**：
  - 电脑：方向键 或 WASD 键
  - 手机：触屏滑动 或 虚拟方向按钮
- **三种难度模式**：
  - 简单模式：速度较慢（150ms）
  - 中等模式：速度适中（100ms）- 默认
  - 困难模式：速度较快（60ms）

### 🚧 特色障碍系统
- **智能生成**：游戏开始 **15秒后** 自动生成障碍物
- **定时更新**：每 **20秒** 刷新一次障碍物位置
- **提前预警**：障碍物生成前 **5秒** 开始倒计时提示
- **复杂形状**：障碍物以多种复杂形状生成（直线、L形、2x2方块、T形等）
- **逻辑严谨**：
  - 仅在游戏进行时自动生成障碍
  - 游戏结束或未开始时不会生成
  - 暂停游戏时暂停障碍生成倒计时

### 🎨 界面设计
- **科技感配色**：深蓝背景配合霓虹色（青蓝、洋红、翠绿）
- **响应式布局**：完美适配各种屏幕尺寸（手机、平板、电脑）
- **横屏/竖屏**：支持横屏和竖屏显示，游戏场地自动调整
- **视觉效果**：
  - 渐变色边框和发光效果
  - 蛇身渐变和头部眼睛动画
  - 食物和障碍物发光特效
  - 障碍警告脉冲动画

## 🛠️ 技术栈

- **前端框架**：Vue 3 (Composition API)
- **编程语言**：JavaScript
- **样式框架**：Tailwind CSS
- **UI 组件库**：Vuetify 3
- **路由管理**：Vue Router 4
- **构建工具**：Vite
- **图标库**：Material Design Icons (MDI)

## 🎯 核心组件说明

### SnakeGame.vue
游戏的核心组件，包含完整的游戏逻辑：

- **游戏状态管理**：idle（空闲）、playing（游戏中）、paused（暂停）、gameover（游戏结束）
- **Canvas 渲染**：使用 HTML5 Canvas 进行高性能游戏渲染
- **碰撞检测**：墙壁碰撞、自身碰撞、障碍物碰撞
- **食物生成**：智能避开蛇身和障碍物
- **障碍物系统**：
  - `generateObstacles()`：生成复杂形状的障碍物
  - `scheduleNextObstacle()`：调度下一次障碍生成
  - `OBSTACLE_FIRST_DELAY`：首次障碍延迟（15秒）
  - `OBSTACLE_INTERVAL`：障碍更新间隔（20秒）
  - `OBSTACLE_WARNING_TIME`：提前警告时间（5秒）
- **响应式适配**：`handleResize()` 自动调整画布大小

## 🚀 快速开始

### 环境要求
- Node.js 16+ 
- npm 或 yarn 或 pnpm

### 安装依赖
```bash
cd snake-game
npm install
```

### 开发模式
```bash
npm run dev
```

然后在浏览器中打开 `http://localhost:5173` 即可开始游戏。

### 构建生产版本
```bash
npm run build
```

构建产物将生成在 `dist` 目录中。

### 预览生产构建
```bash
npm run preview
```

## 🎨 设计理念

### 色彩方案
采用赛博朋克/科技感配色：
- **背景色**：深蓝色渐变 (#0a0a1a → #1a1a2e)
- **主色调**：青蓝色 (#00d4ff) - 用于边框和UI元素
- **强调色**：洋红色 (#ff00ff) - 用于食物和次要强调
- **成功色**：翠绿色 (#00ff88) - 用于蛇头和成功状态
- **警告色**：橙色 (#ffaa00) - 用于倒计时提示
- **错误色**：玫红色 (#ff3366) - 用于障碍物

### 布局策略
- **响应式容器**：使用 Vuetify 的 grid 系统
- **自适应场地**：游戏场地根据可用空间自动调整大小，最大 600x600px
- **移动端优化**：
  - 小屏幕下显示虚拟方向按钮
  - 触摸滑动支持
  - 按钮和文字大小自动适配

## 📝 注意事项

1. **障碍生成逻辑**：
   - 障碍物仅在游戏进行状态（playing）时自动生成
   - 暂停游戏会暂停所有计时器
   - 重置游戏会清除所有障碍物和计时器

2. **碰撞检测**：
   - 障碍物不会生成在蛇头附近（5格安全距离）
   - 食物不会生成在蛇身或障碍物上
   - 障碍物以多种复杂形状随机生成，增加游戏趣味性

3. **性能优化**：
   - 使用 Canvas 进行高性能渲染
   - 游戏循环使用 setTimeout 精确控制速度
   - 组件卸载时自动清除所有计时器
