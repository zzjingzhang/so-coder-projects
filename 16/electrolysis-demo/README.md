# 电解水实验演示

一个用于高中化学教学的电解水实验演示网页应用，通过动画展示水分子分解成氢气和氧气的完整过程。

## 项目简介

本项目是一个交互式的化学教学辅助工具，旨在帮助学生理解电解水的化学原理。通过生动的SVG动画、分步骤展示和详细的知识点讲解，使抽象的化学概念变得直观易懂。

### 主要功能

- **动态动画展示**：通过SVG动画实时展示电解水装置的工作过程
- **分步骤讲解**：5个关键步骤详细展示电解水的完整过程
- **化学方程式**：每个步骤都配有对应的化学反应方程式
- **知识点详解**：深入解释每个步骤涉及的化学原理
- **思考问题**：5个精选问题及详细解答，帮助学生加深理解
- **自动播放**：支持自动按顺序播放所有步骤
- **响应式设计**：适配各种屏幕尺寸

## 核心功能模块

### 1. 电解水动画模块

- 使用SVG绘制完整的电解装置图
- 动态气泡动画（氢气气泡较多，氧气气泡较少但较大）
- 水位变化动画
- 电流流动效果
- 电极发光效果

### 2. 分步骤展示模块

5个关键步骤：
1. **准备阶段**：装置介绍、电解质作用
2. **接通电源**：电流路径、直流电原理
3. **阴极反应**：还原反应、氢气生成
4. **阳极反应**：氧化反应、氧气生成
5. **总反应与结论**：体积比2:1、水的组成

### 3. 思考问题模块

5个深入问题：
- 为什么需要加入电解质？
- 氢气和氧气体积比为什么是2:1？
- 阴极和阳极分别发生什么反应？
- 电解水是吸热还是放热反应？
- 如何验证产生的气体？

## 使用说明

### 操作方式

1. **手动步骤控制**：
   - 点击「上一步」/「下一步」按钮切换步骤
   - 点击顶部步骤条可直接跳转到任意步骤

2. **自动播放**：
   - 点击「自动播放」按钮，系统将自动按顺序播放所有步骤
   - 每步间隔3秒，可随时点击「暂停」停止

3. **查看思考问题**：
   - 点击问题标题展开查看详细解答
   - 支持同时展开多个问题

### 化学知识点

#### 电解水反应方程式

- **阴极（还原反应）**：`2H₂O + 2e⁻ → H₂↑ + 2OH⁻`
- **阳极（氧化反应）**：`2H₂O → O₂↑ + 4H⁺ + 4e⁻`
- **总反应**：`2H₂O → 2H₂↑ + O₂↑`

#### 关键结论

1. 氢气与氧气的体积比 = 2 : 1
2. 水是由氢元素和氧元素组成的化合物
3. 化学变化中分子可分，原子不可分
4. 电解水是吸热反应，需要持续供电

## 教学建议

1. **配合实际实验**：建议在实际实验操作前或操作后使用本演示工具
2. **重点讲解**：
   - 步骤3和步骤4（阴极和阳极反应）
   - 氢气和氧气体积比2:1的含义
3. **互动提问**：使用思考问题模块引导学生思考
4. **拓展讨论**：可拓展讨论电解水在氢能领域的应用
