📝 电解水反应总结

阴极反应（还原）

2H₂O + 2e⁻ → H₂↑ + 2OH⁻

水分子获得电子，被还原生成氢气

阳极反应（氧化）

2H₂O → O₂↑ + 4H⁺ + 4e⁻

水分子失去电子，被氧化生成氧气

总反应方程式

2H₂O → 2H₂↑ + O₂↑

体积比: H₂ : O₂ = 2 : 1
