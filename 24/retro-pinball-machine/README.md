# Retro Pinball Machine - 复古弹球机

一款基于 Vue 3 开发的复古风格弹球机游戏，使用 Canvas 2D 实现真实物理引擎，包含翻转控制、保险杠弹跳、得分乘数、多球模式和霓虹灯闪光灯效果。

## 项目结构

```
retro-pinball-machine/
├── src/
│   ├── views/
│   │   ├── Home.vue          # 首页视图
│   │   └── PinballGame.vue   # 弹球机游戏视图（包含完整游戏逻辑）
│   ├── router/
│   │   └── index.js          # Vue Router 路由配置
│   ├── App.vue               # 根组件
│   ├── main.js               # 应用入口
│   └── style.css             # 全局样式（包含 Tailwind 和自定义复古风格）
├── index.html                # HTML 模板
├── package.json              # 项目依赖配置
├── vite.config.js            # Vite 配置
├── tailwind.config.js        # Tailwind CSS 配置
└── postcss.config.js         # PostCSS 配置
```

## 技术栈

- **框架**: Vue 3 (Composition API)
- **语言**: JavaScript
- **样式**: Tailwind CSS
- **UI 组件库**: Element Plus
- **路由**: Vue Router 4
- **打包工具**: Vite
- **游戏渲染**: HTML5 Canvas 2D

## 功能特性

### 核心游戏功能
- **翻转控制**: 使用左右方向键控制挡板，精准弹射弹球
- **真实物理引擎**: 包含重力模拟、碰撞检测、速度限制
- **保险杠弹跳**: 8 个彩色保险杠，碰撞时有弹跳力加成和得分
- **墙壁碰撞**: 弹球与边界墙壁的物理反弹效果

### 得分系统
- **基础得分**: 每个保险杠碰撞获得 100 分
- **得分乘数**: 连续 3 次以上碰撞增加得分乘数（最高 x5）
- **连击计数**: 显示当前连击次数，2秒无操作后重置
- **弹球物理轨迹**: 弹球带有发光拖尾效果，视觉更炫酷

### 高级特性
- **多球模式**: 得分达到 10000 分时自动触发多球模式（同时 3 个弹球）
- **蓄力发射**: 长按空格键蓄力，松开发射，蓄力时间影响发射力度
- **暂停功能**: 按 P 键或点击暂停按钮暂停游戏
- **闪光灯效果**: 碰撞保险杠时产生扩散的霓虹灯闪光效果

### 视觉效果
- **复古霓虹风格**: 粉色、青色、黄色、橙色、绿色等霓虹色彩
- **网格背景**: 深色背景搭配半透明网格
- **脉冲动画**: 保险杠持续脉冲发光，碰撞时发光增强
- **视觉响应**: 挡板激活状态有进度条反馈，发射蓄力有视觉提示

## 操作说明

| 操作 | 按键 | 说明 |
|------|------|------|
| 左挡板 | ← 左方向键 | 按住激活，松开复位 |
| 右挡板 | → 右方向键 | 按住激活，松开复位 |
| 发射弹球 | 空格键 | 长按蓄力，松开发射 |
| 暂停/继续 | P 键 | 暂停或继续游戏 |

### 游戏规则
1. 每局游戏有 3 个弹球
2. 撞击保险杠获得分数
3. 连续撞击增加得分乘数
4. 得分达到 10000 分触发多球模式
5. 弹球从底部掉落则丢失一个弹球
6. 3 个弹球全部丢失则游戏结束

## 安装与启动

### 环境要求
- Node.js >= 16.0.0
- npm 或 yarn

### 安装依赖

```bash
cd retro-pinball-machine
npm install
```

### 开发模式启动

```bash
npm run dev
```

项目启动后，在浏览器中访问显示的本地地址（通常是 `http://localhost:5173`）即可开始游戏。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

## 游戏技术实现

### 物理引擎
- **重力加速度**: 0.15 单位/帧
- **弹跳阻尼**: 0.85（碰撞后速度衰减）
- **挡板力**: 15 单位力
- **最大速度**: 25 单位/帧（速度限制防止过快）

### 碰撞检测
- 圆形与圆形碰撞（弹球与保险杠）
- 圆形与线段碰撞（弹球与挡板）
- 圆形与矩形边界碰撞（弹球与墙壁）
- 最近点计算用于线段碰撞检测

### 游戏循环
使用 `requestAnimationFrame` 实现 60fps 的游戏循环，包含：
1. 背景绘制
2. 墙壁绘制
3. 保险杠更新与绘制
4. 挡板更新与绘制
5. 弹球更新、碰撞检测与绘制
6. 闪光特效更新与绘制

## 自定义扩展

### 修改保险杠位置
在 `PinballGame.vue` 的 `initGame()` 函数中修改 `bumperPositions` 数组。

### 调整物理参数
修改 `GRAVITY`、`BOUNCE_DAMPING`、`FLIPPER_FORCE`、`BUMPER_BOUNCE` 等常量值。

### 添加更多颜色
在 `COLORS.bumper` 数组中添加更多霓虹色选项。

## 开发说明

- 游戏逻辑全部封装在 `PinballGame.vue` 中，使用类的方式组织代码
- 响应式状态使用 Vue 3 的 `ref`
- 键盘事件使用 `window.addEventListener` 监听
- 生命周期使用 `onMounted` 和 `onUnmounted` 管理
