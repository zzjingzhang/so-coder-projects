# 富文本表格编辑器

这是一个功能强大的富文本编辑器，支持高级表格编辑功能，包括插入、删除、合并单元格以及表格样式自定义。

## 技术栈

- **React**: 18.3.1
- **Vite**: 8.x
- **Tailwind CSS**: 4.x
- **PrimeReact**: 10.8.4
- **PrimeIcons**: 7.0.0
- **React Router**: 6.28.0
- **PostCSS**: 8.x
- **Autoprefixer**: 10.x

## 主要功能

### 富文本编辑功能
- 文本格式化：加粗、斜体、下划线
- 文本对齐：左对齐、居中对齐、右对齐
- 列表功能：无序列表、有序列表

### 表格编辑功能
- **插入表格**：可自定义行数、列数、边框样式等
- **行操作**：
  - 在当前行上方插入新行
  - 在当前行下方插入新行
  - 删除当前行
- **列操作**：
  - 在当前列左侧插入新列
  - 在当前列右侧插入新列
  - 删除当前列
- **单元格操作**：
  - 合并选中的单元格
  - 拆分已合并的单元格
- **表格删除**：删除整个表格

### 表格样式自定义
- **边框样式**：
  - 自定义边框颜色
  - 自定义边框宽度（0-10px）
- **单元格样式**：
  - 自定义单元格背景色
  - 自定义文字颜色
- **对齐方式**：
  - 左对齐
  - 居中对齐
  - 右对齐

### 交互体验
- **弹出层工具栏**：点击表格单元格时，会在单元格上方显示操作工具栏
- **实时预览**：样式更改实时应用到表格
- **直观操作**：使用 PrimeReact 组件库提供的美观界面

## 安装和运行

### 前提条件
- Node.js 16.x 或更高版本
- npm 7.x 或更高版本

### 安装依赖

```bash
cd rich-text-table-editor
npm install
```

### 开发模式

```bash
npm run dev
```

这将启动开发服务器，默认在 `http://localhost:5173` 运行。

### 构建生产版本

```bash
npm run build
```

构建后的文件将输出到 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

## 使用说明

### 访问应用
启动开发服务器后，访问 `http://localhost:5173` 即可看到应用首页。

### 导航
- **首页**：应用介绍和功能说明
- **编辑器**：富文本编辑器界面
- **关于**：技术栈和功能介绍

### 使用编辑器

1. **基础文本编辑**：
   - 使用工具栏按钮进行文本格式化
   - 支持加粗、斜体、下划线等格式
   - 支持对齐方式和列表

2. **插入表格**：
   - 点击工具栏中的表格图标
   - 在弹出的对话框中设置表格参数：
     - 行数（1-20）
     - 列数（1-10）
     - 边框颜色
     - 边框宽度
     - 单元格背景色
     - 文字颜色
     - 对齐方式
   - 点击"插入"按钮

3. **编辑表格**：
   - 点击表格中的任意单元格
   - 会在单元格上方显示表格工具栏
   - 使用工具栏中的按钮进行操作：
     - **行操作**：上插行、下插行、删除行
     - **列操作**：左插列、右插列、删除列
     - **单元格操作**：合并、拆分
     - **样式设置**：边框颜色、边框宽度、背景色、文字颜色、对齐方式
     - **应用/删除**：应用样式、删除表格

4. **合并单元格**：
   - 选中要合并的多个单元格
   - 点击工具栏中的"合并"按钮

5. **拆分单元格**：
   - 点击已合并的单元格
   - 点击工具栏中的"拆分"按钮

6. **自定义样式**：
   - 点击表格单元格显示工具栏
   - 使用颜色选择器设置边框颜色、背景色、文字颜色
   - 使用数字输入框设置边框宽度
   - 使用下拉菜单选择对齐方式
   - 点击"应用样式"按钮应用更改

## 技术细节

### 富文本编辑实现
- 使用 HTML5 的 `contentEditable` 属性实现编辑区域
- 使用 `document.execCommand` 执行编辑命令
- 支持撤销/重做操作

### 表格操作实现
- 使用 DOM 操作实现表格的增删改
- 支持 `rowSpan` 和 `colSpan` 属性实现单元格合并
- 使用 `getBoundingClientRect` 计算工具栏位置

### 样式系统
- 使用 Tailwind CSS 进行响应式布局
- 使用 PrimeReact 组件库提供 UI 组件
- 支持自定义主题和样式
