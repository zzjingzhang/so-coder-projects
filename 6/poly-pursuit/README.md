# PolyPursuit (多边形追逐)

一款低多边形风格的平台动作收集游戏，在几何世界中探索、收集、挑战难关。

## 游戏特色

- **低多边形艺术风格**：所有角色、环境均由鲜明的多边形构成，高饱和度色彩营造独特视觉效果
- **3个独特关卡**：从晶体山谷到熔岩深渊，再到天空堡垒，每一关都有独特的挑战
- **平台动作玩法**：跑、跳、躲避敌人，穿越各种地形障碍
- **收集系统**：收集能量碎片解锁新关卡，寻找稀有遗物获得高分
- **自动存档**：游戏进度自动保存，无需担心丢失进度

## 技术栈

- **前端框架**: React 19
- **开发语言**: TypeScript
- **样式方案**: Tailwind CSS
- **UI 组件库**: Ant Design
- **路由管理**: React Router
- **构建工具**: Vite

## 游戏操作

| 按键 | 功能 |
|------|------|
| ← → / A D | 左右移动 |
| ↑ / W / 空格 | 跳跃 |
| ESC / P | 暂停游戏 |
| R | 重新开始（游戏结束后） |

## 游戏机制

### 角色控制
- 玩家控制一个六边形角色，可左右移动和跳跃
- 角色有 3 点生命值，生命值耗尽则游戏结束

### 收集物品
- **能量碎片（黄色五边形）**：收集后增加分数，解锁新关卡
- **稀有遗物（彩色八边形）**：收集后获得大量分数

### 敌人
- 三角形敌人会在平台上来回巡逻
- 从敌人头顶跳跃可以消灭敌人并获得分数
- 被敌人侧面碰撞会损失生命值

### 平台
- 普通平台：固定位置的平台
- 移动平台：会来回移动的平台，边缘有虚线标识

### 传送门
- 每关末尾都有传送门，进入后进入下一关
- 最后一关的传送门会结束游戏并显示胜利画面

## 关卡介绍

### 关卡 1：晶体山谷
- 难度：简单
- 背景：深蓝色调，充满神秘晶体的山谷
- 特色：学习基础操作，熟悉跳跃和收集

### 关卡 2：熔岩深渊
- 难度：中等
- 背景：红棕色调，危险的熔岩地带
- 特色：断裂的地面，更多移动平台和敌人

### 关卡 3：天空堡垒
- 难度：困难
- 背景：深蓝紫色调，云端之上的古老堡垒
- 特色：复杂的平台布局，多个敌人，最终挑战

## 开发说明

### 游戏引擎
游戏使用 HTML5 Canvas 进行渲染，核心游戏逻辑位于 `src/hooks/useGameEngine.ts`，包含：
- 物理系统（重力、碰撞检测）
- 输入处理
- 游戏状态管理
- 渲染系统

### 状态管理
使用 React Context (`src/context/GameContext.tsx`) 管理全局游戏状态，包括：
- 总得分
- 收集的能量数量
- 已解锁关卡
- 游戏设置

### 数据持久化
游戏进度会自动保存到浏览器的 localStorage 中，包括：
- 总得分
- 收集的能量
- 已解锁关卡
- 游戏设置
