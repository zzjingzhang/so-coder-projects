# 像素机器人

一个像素风格的益智游戏，玩家需要通过放置方向箭头来引导机器人避开障碍物、收集金币并到达终点。

## 游戏简介

- **游戏标题**: 像素机器人
- **核心玩法**: 在网格地图上放置方向箭头，引导自动前进的机器人从起点到达终点
- **艺术风格**: 像素艺术风格，鲜明对比的色彩，简单的逐帧动画
- **目标受众**: 喜欢策略规划和轻度益智挑战的各年龄段玩家

## 技术栈

- **框架**: React 19
- **语言**: JavaScript
- **样式**: Tailwind CSS 4
- **路由**: React Router 7
- **打包工具**: Vite 8

## 游戏特性

### 核心机制
1. **自动前进机器人**: 机器人默认沿初始方向直线前进，无法停止
2. **方向箭头**: 玩家在机器人行动前，在地图格子上放置方向箭头
3. **方向改变**: 机器人走到有箭头的格子时，会立即按照箭头指示的方向改变行进方向
4. **箭头数量限制**: 部分关卡有箭头数量限制
5. **失败条件**: 碰到障碍物或掉落地图边缘则游戏失败

### 游戏元素
- **🤖 机器人**: 自动前进的主角，跟随箭头方向移动
- **🏁 终点**: 引导机器人到达此处即可过关
- **💰 金币**: 可选收集目标，增加游戏乐趣
- **🧱 障碍物**: 碰到即失败的危险区域
- **↑↓←→ 方向箭头**: 玩家放置的方向指示器

### 关卡设计
游戏包含 5 个精心设计的关卡：
1. **初次见面**: 学习基本操作，引导机器人到达终点
2. **转弯学习**: 学习放置箭头改变机器人方向
3. **躲避障碍**: 避开障碍物到达终点
4. **迷宫探索**: 在复杂地图中找到路径
5. **极限挑战**: 最少箭头，最多障碍

## 安装和运行

### 环境要求
- Node.js >= 16.0.0
- npm >= 7.0.0

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

开发服务器将启动，通常在 `http://localhost:5173` 或附近端口。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

## 游戏操作说明

1. **选择关卡**: 在关卡选择页面点击想要挑战的关卡
2. **观察地图**: 注意观察起点、终点、障碍物和金币位置
3. **选择箭头方向**: 点击右侧的方向按钮（↑↓←→）选择要放置的箭头方向
4. **放置箭头**: 在地图上的可用格子上点击，放置所选方向的箭头
5. **移除箭头**: 右键点击已放置的箭头可以移除它，或者在未选择方向时左键点击
6. **开始游戏**: 点击"开始游戏"按钮，机器人将开始自动移动
7. **观看结果**: 观看机器人运行，成功则进入下一关，失败则点击"重置"重新调整箭头

## 游戏策略提示

1. **规划路径**: 在放置箭头之前，先在脑海中规划好机器人的完整路径
2. **节约箭头**: 注意关卡的箭头数量限制，合理规划每个箭头的位置
3. **收集金币**: 尝试收集所有金币以获得更好的游戏体验
4. **避免死路**: 确保机器人不会走进死胡同或撞到障碍物
5. **多次尝试**: 不要害怕失败，重置后重新规划路径

## 自定义关卡

如果想要添加或修改关卡，可以编辑 `src/data/levels.js` 文件。每个关卡包含以下属性：

- `id`: 关卡唯一标识
- `name`: 关卡名称
- `description`: 关卡描述
- `gridSize`: 地图大小（rows x cols）
- `startPos`: 起点位置 {row, col}
- `endPos`: 终点位置 {row, col}
- `startDirection`: 机器人初始方向
- `maxArrows`: 最大箭头数量（0 表示无限制）
- `obstacles`: 障碍物位置数组
- `coins`: 金币位置数组
- `specialArrows`: 特殊箭头数组（预留功能）

## 开发说明

### 项目使用的主要依赖

- `react`: ^19.2.5 - React 框架
- `react-dom`: ^19.2.5 - React DOM 渲染
- `react-router-dom`: ^7.14.2 - React 路由
- `tailwindcss`: ^4.2.4 - Tailwind CSS 样式框架
- `postcss`: ^8.5.12 - CSS 处理工具
- `autoprefixer`: ^10.5.0 - CSS 自动前缀
- `vite`: ^8.0.10 - 构建工具

### 代码架构

- **类型定义**: `src/types/index.js` 定义了游戏中使用的所有常量和枚举
- **游戏逻辑**: `src/utils/gameUtils.js` 包含游戏核心逻辑函数
- **状态管理**: `src/hooks/useGameState.js` 使用 React Hooks 管理游戏状态
- **页面组件**: `src/pages/` 包含所有页面组件
- **关卡数据**: `src/data/levels.js` 包含所有关卡配置
- **样式**: `src/index.css` 定义了像素风格的全局样式和动画
