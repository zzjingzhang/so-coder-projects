# Todo 列表 Web 应用

一个功能完整的 Todo 任务管理应用，支持用户注册、登录，以及任务的增删改查功能。

## 功能特性

- 🔐 **用户认证**：支持用户注册、登录、退出登录
- ✅ **任务管理**：支持任务的创建、查看、编辑、删除（CRUD 操作）
- 🎨 **自定义样式**：每个任务支持自定义卡片背景颜色、文字颜色和字体
- 🔍 **搜索筛选**：支持按关键词搜索任务，按状态和优先级筛选
- 📊 **任务统计**：首页展示任务统计数据（总任务数、已完成、进行中、待处理）
- 📱 **响应式布局**：完美适配桌面端和移动端

## 页面说明

| 路由 | 页面 | 说明 |
|------|------|------|
| `/login` | 登录页面 | 用户登录界面 |
| `/register` | 注册页面 | 用户注册界面 |
| `/` | 首页/任务列表 | 展示所有任务，支持搜索和筛选 |
| `/create` | 创建任务 | 新建任务表单，支持实时预览 |
| `/edit/:id` | 编辑任务 | 修改已有任务的表单 |

## 数据存储

项目使用浏览器的 `localStorage` 进行数据持久化存储：

- **用户数据**：存储在 `users` 键中
- **当前登录用户**：存储在 `currentUser` 键中
- **任务数据**：每个用户的任务存储在 `todos_{userId}` 键中

## 安装与运行

### 环境要求

- Node.js >= 16.0.0
- npm >= 7.0.0

### 安装依赖

```bash
cd todo-app
npm install
```

### 开发模式运行

```bash
npm run dev
```

项目启动后，访问 http://localhost:5173 查看应用。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

## 使用说明

### 1. 注册账号

1. 打开应用，自动跳转到登录页面
2. 点击「立即注册」链接
3. 填写用户名、邮箱、密码，完成注册

### 2. 登录账号

1. 在登录页面输入邮箱和密码
2. 点击「登录」按钮
3. 登录成功后自动跳转到任务列表页面

### 3. 创建任务

1. 点击顶部导航栏的「新建任务」按钮
2. 填写任务标题（必填）、描述（可选）
3. 选择优先级、状态、截止日期
4. 自定义卡片背景颜色、文字颜色、字体样式（右侧可实时预览效果）
5. 点击「创建任务」按钮

### 4. 管理任务

在任务列表页面可以：

- **搜索**：在搜索框中输入关键词搜索任务标题或描述
- **筛选**：按状态或优先级筛选任务
- **编辑**：点击任务卡片上的「编辑」按钮修改任务
- **删除**：点击任务卡片上的「删除」按钮，确认后删除任务

### 5. 自定义样式

创建或编辑任务时，可以：

- **卡片背景颜色**：从 15 种预设颜色中选择
- **文字颜色**：从 12 种预设颜色中选择
- **字体样式**：从 8 种字体中选择（默认字体、宋体、黑体、微软雅黑、Arial、Times New Roman、Courier New、Georgia）

## 特性亮点

1. **实时预览**：创建/编辑任务时，右侧实时显示卡片样式效果
2. **路由保护**：未登录用户无法访问任务相关页面，自动跳转登录页
3. **响应式设计**：适配从手机到桌面的各种屏幕尺寸
4. **友好提示**：使用 Ant Design 的 Message 组件提供操作反馈
5. **数据持久化**：刷新页面或关闭浏览器后，数据不会丢失

## 开发说明

### 状态管理

项目使用 React Context 进行状态管理：

- `AuthContext`：管理用户认证状态（登录、注册、退出）
- `TodoContext`：管理任务数据（增删改查）

### 组件设计

- 采用模块化设计，每个页面和组件独立
- 使用 Tailwind CSS 进行样式开发
- 结合 Ant Design 组件库快速构建 UI
