一款高端科技感的智能贷款计算器，支持多种贷款参数的互相计算。

## 功能特性

### 输入参数
- 💰 **贷款总额** (元) - 贷款本金金额
- 📅 **月供金额** (元) - 每月还款金额
- 📊 **月息** (%) - 月利率
- 📈 **年息** (%) - 年利率
- 🎯 **综合年利率 (APR)** (%) - 年度化综合利率
- ⏰ **贷款期限** (月) - 贷款总月数

### 计算功能
- **互相计算**: 支持根据已知参数计算未知参数
- **自动换算**: 月息和年息自动互相换算
- **智能提示**: 输入不足时给出友好提示

### 操作按钮
- 🖥️ **开始计算** - 执行计算逻辑
- 🔄 **重置** - 清空所有输入
- ❌ **关闭程序** - 关闭当前浏览器窗口

## 视觉设计

- **高端科技感**: 深色主题配合渐变色彩
- **彩色风格**: 紫色、青色、粉色、黄色等多种渐变配色
- **动态效果**: 背景动画光晕、按钮悬停效果
- **响应式布局**: 支持移动端和桌面端

## 如何启动项目

### 环境要求
- Node.js >= 18.0.0
- npm >= 9.0.0

### 安装依赖

```bash
cd loan-calculator
npm install
```

### 开发模式

```bash
npm run dev
```

项目将在 `http://localhost:5173` 启动（如果端口被占用，会自动切换到其他端口）。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

### 代码检查

```bash
npm run lint
```

## 使用说明

1. **计算月供**: 输入【贷款总额】、【利率】、【贷款期限】，点击"开始计算"
2. **计算贷款总额**: 输入【月供金额】、【利率】、【贷款期限】，点击"开始计算"
3. **计算利率**: 输入【贷款总额】、【月供金额】、【贷款期限】，点击"开始计算"
4. **月息和年息**: 输入其中一个，另一个会在计算时自动换算

## 计算公式

### 月供计算公式（等额本息）
```
月供 = 贷款总额 × (月利率 × (1 + 月利率)^贷款期限) / ((1 + 月利率)^贷款期限 - 1)
```

### 月息与年息换算
```
月息 = 年息 / 12
年息 = 月息 × 12
```
