# 📚 英语单词拼写练习工具

一个专为小学生设计的英语单词拼写练习网页应用，让孩子们在有趣的交互中掌握英语单词拼写。

## ✨ 功能特性

- 📝 **单词导入** - 支持粘贴文本或导入 Excel 文件
- ⏰ **时间设置** - 可选择 3/5/10/15/20 分钟的练习时长
- 🎯 **拼写练习** - 根据中文提示，一个字母一个字母地拼写英文单词
- 💡 **智能提示** - 拼错时有提示，可重新尝试，支持首字母提示
- 📊 **结果统计** - 练习结束后显示正确率、用时、错题列表
- 📱 **响应式设计** - 完美适配平板电脑和各种屏幕尺寸

## 🛠️ 技术栈

- **框架**: React 19
- **语言**: JavaScript
- **样式**: Tailwind CSS 4
- **UI 组件库**: Ant Design 6
- **路由**: React Router 7
- **构建工具**: Vite 8
- **Excel 处理**: xlsx

## 📖 使用指南

### 1. 导入单词

有两种方式导入单词：

**方式一：粘贴文本**
- 每行一个单词
- 格式：`英文单词,中文释义`（英文在前，中文在后，用逗号或制表符分隔）
- 示例：
```
apple,苹果
banana,香蕉
cat,猫
dog,狗
```

**方式二：导入 Excel**
- 支持 .xlsx, .xls, .csv 格式
- 第一列为英文单词，第二列为中文释义
- 第一行可以是表头（会自动跳过）

### 2. 设置练习

- 选择练习时间：3分钟、5分钟、10分钟、15分钟、20分钟
- 确认单词数量后点击"开始练习"

### 3. 开始练习

- 屏幕会显示中文释义
- 在下方的输入框中一个字母一个字母地拼写出对应的英文单词
- 每个输入框只能输入一个字母
- 输完最后一个字母后会自动判断对错

**功能按钮：**
- 💡 获取提示：显示单词首字母
- ⏸️ 暂停/继续：暂停或恢复练习
- ❌ 退出：退出当前练习

### 4. 查看结果

练习结束后会显示：
- 🏆 表现评级（优秀、良好、继续加油、需要多练习）
- ✅ 答对题数
- ❌ 答错题数
- 📊 正确率
- ⏱️ 用时
- 📝 需要复习的单词列表（错题）
- ✨ 已掌握的单词列表

## 🔧 开发说明

### 状态管理

使用 React Context (`PracticeContext.jsx`) 管理全局状态：
- `words` - 单词列表
- `practiceTime` - 练习时间（分钟）
- `currentIndex` - 当前单词索引
- `correctCount` - 答对数量
- `incorrectWords` - 错题列表
- `startTime/endTime` - 练习开始/结束时间

### 路由配置

```
/          -> 首页
/import    -> 导入单词页面
/setup     -> 练习设置页面
/practice  -> 拼写练习页面
/result    -> 结果展示页面
```
