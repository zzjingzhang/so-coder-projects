# 溶液浓度可视化演示

一个用于高中化学课堂的溶液浓度教学工具，通过可视化方式展示溶液混合时浓度的变化效果。

## 项目简介

本项目是一个交互式的溶液浓度可视化演示工具，旨在帮助学生直观理解溶液浓度的概念。学生可以通过调整两个烧杯中的溶质质量和溶剂体积，观察液体高度和颜色的变化（颜色深浅反映溶液浓度），然后模拟混合后的结果。

## 功能特性

- 🧪 **双烧杯可视化**：直观展示两个独立的溶液烧杯
- 🎨 **颜色浓度映射**：浓度越高，颜色越深，直观呈现浓度变化
- 📏 **体积高度对应**：液体体积越大，液面高度越高
- 🎚️ **滑动条控制**：通过滑动条精确调整溶质质量和溶剂体积
- ➕ **混合模拟**：一键混合两种溶液，显示混合后的浓度和计算详情
- 🔄 **重置功能**：快速重置到初始状态
- 📊 **计算详情展示**：展示完整的浓度计算过程和公式

## 浓度计算公式

溶液质量分数浓度的计算公式：

```
浓度 = 溶质质量 ÷ (溶质质量 + 溶剂质量) × 100%
```

> **注意**：本演示中假设溶剂（水）的密度为 1g/mL，因此体积(mL) 近似等于质量(g)。

## 安装与运行

### 环境要求

- Node.js >= 16.0.0
- npm >= 7.0.0

### 安装依赖

```bash
cd solution-concentration-visualizer
npm install
```

### 开发模式

```bash
npm run dev
```

项目启动后，访问浏览器显示的本地地址（如 `http://localhost:5173/`）即可查看应用。

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 使用说明

1. **调整溶液参数**：
   - 使用左侧滑动条调整「溶液 A」的溶质质量（0-50g）和溶剂体积（10-100mL）
   - 使用右侧滑动条调整「溶液 B」的溶质质量（0-50g）和溶剂体积（10-100mL）

2. **观察变化**：
   - 烧杯中液体的高度会随总容积变化
   - 液体的颜色会随浓度变化（浓度越高，颜色越深）
   - 烧杯下方实时显示当前溶液浓度

3. **混合溶液**：
   - 点击中间的「混合两种溶液」按钮
   - 下方会显示混合结果区域，包含：
     - 混合后溶液的可视化展示
     - 混合后的浓度数值
     - 详细的计算过程表格
     - 浓度计算式

4. **重置**：
   - 点击「重置」按钮可将所有参数恢复到初始值

## 教学建议

本工具可用于以下教学场景：

1. **溶液浓度概念引入**：通过调整参数，让学生直观感受浓度与溶质、溶剂的关系
2. **稀释问题演示**：固定溶质质量，增加溶剂体积，观察浓度降低
3. **混合问题计算**：混合两种不同浓度的溶液，验证计算结果
4. **饱和溶液讨论**：讨论浓度上限和溶解度概念
