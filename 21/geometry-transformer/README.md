# 几何图形变换工具

这是一个专为小学数学教学设计的交互式几何图形变换工具，帮助学生直观理解图形的平移和旋转概念。

## 功能特性

### 🎨 图形选择
- 支持多种预设几何图形：三角形、矩形、正方形、五边形、六边形
- 每种图形使用不同的颜色标识，便于区分

### 🔄 旋转控制
- 滑块精确控制旋转角度（0°-360°）
- 快捷按钮：±45°、±90°
- 实时显示当前旋转角度
- 支持选择不同的旋转中心点

### ➡️ 平移控制
- 方向按钮：上下左右快速平移
- 滑块精确控制水平和垂直平移
- 实时显示当前平移坐标

### 📍 旋转中心点
- 多种预设选项：图形中心、四个角落
- 支持自定义中心点坐标
- 中心点在画布上以红色十字标识

### 📊 轨迹保存
- 保存变换后的图形状态
- 显示历史轨迹，帮助理解连续变换
- 可调节轨迹透明度
- 一键清除历史记录

### ✏️ 自定义图形
- 学生可以在画布上点击创建自定义图形
- 支持任意多边形（至少3个顶点）
- 实时预览正在创建的图形
- 完成后自动应用所有变换功能

## 安装和运行

### 前置要求

- Node.js (建议版本 18.x 或更高)
- npm (随 Node.js 一起安装)
- Angular CLI (建议版本 21.x)

### 安装步骤

1. **进入项目目录**
   ```bash
   cd geometry-transformer
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动开发服务器**
   ```bash
   npm start
   ```

4. **访问应用**
   
   打开浏览器，访问 `http://localhost:4200`

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist/` 目录。

## 使用指南

### 选择图形

1. 在右侧控制面板的"选择图形"区域
2. 点击预设图形按钮（三角形、矩形、正方形、五边形、六边形）
3. 或点击"创建自定义图形"按钮来自定义图形

### 旋转图形

1. 使用滑块精确调整旋转角度
2. 或使用快捷按钮（±45°、±90°）
3. 当前角度会实时显示

### 平移图形

1. 使用方向按钮快速平移
2. 或使用滑块精确调整
3. 当前坐标会实时显示

### 选择旋转中心点

1. 在"旋转中心点"下拉菜单中选择
2. 选择"自定义"时可输入具体坐标
3. 中心点会以红色十字在画布上标识

### 保存轨迹

1. 调整到满意的变换状态
2. 点击"保存当前状态"按钮
3. 历史状态会以半透明方式显示
4. 可调节轨迹透明度
5. 点击"清除历史轨迹"可清除所有保存的状态

### 创建自定义图形

1. 点击"创建自定义图形"按钮
2. 在画布上点击添加顶点（至少需要3个顶点）
3. 点击"完成创建"按钮确认
4. 新图形将应用所有变换功能

## 设计特点

### 界面设计
- 清晰的区域划分，左侧画布，右侧控制面板
- 色彩搭配合理，不同功能区域使用不同主题色
- 按钮文字水平垂直居中，视觉效果良好
- 响应式设计，支持不同屏幕尺寸

### 教学价值
- 直观的可视化展示，帮助理解抽象概念
- 实时反馈，让学生看到操作的即时效果
- 鼓励探索，支持自定义图形激发创造力
- 轨迹保存功能展示连续变换过程
