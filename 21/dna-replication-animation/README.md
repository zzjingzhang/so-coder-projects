# DNA复制过程交互式动画

一个交互式的DNA复制过程动画演示应用，使用Angular框架开发，展示DNA半保留复制的完整过程。

## 项目简介

本项目是一个教育性的交互式动画应用，旨在帮助用户理解DNA复制的复杂过程。通过直观的动画演示和分步讲解，用户可以清晰地看到：

- 解旋酶如何打开DNA双螺旋结构
- 碱基配对规则（A-T，G-C）如何工作
- DNA聚合酶如何合成新的DNA链
- 半保留复制的完整过程
- 磷酸基团、脱氧核糖和各种碱基的结构

## 技术栈

- **框架**: Angular 21.2.0
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **UI组件库**: NG-ZORRO (Ant Design for Angular)
- **路由**: Angular Router
- **打包工具**: Angular CLI Application Builder
- **测试框架**: Vitest

## 主要功能

### 1. 交互式动画演示
- **分步演示**: 9个关键步骤的完整动画
- **播放控制**: 播放/暂停/上一步/下一步/重置
- **速度调节**: 慢/中/快三档速度
- **步骤导航**: 可直接跳转到任意步骤

### 2. 完整的复制过程
- **初始状态**: 完整的DNA双螺旋结构
- **解旋酶结合**: 解旋酶结合到复制起点
- **DNA解旋**: 双螺旋结构打开，形成复制叉
- **单链稳定**: SSB蛋白稳定单链DNA
- **引物合成**: RNA引物合成
- **DNA聚合酶结合**: 聚合酶准备合成新链
- **碱基配对**: 按照互补配对原则添加核苷酸
- **新链延伸**: 前导链连续合成，后随链冈崎片段
- **复制完成**: 半保留复制结果展示

### 3. 分子结构展示
- **碱基图例**: 清晰的颜色编码（A-红色，T-青色，G-蓝色，C-绿色）
- **分子组件**: 磷酸基团（紫色）、脱氧核糖（蓝色）
- **关键酶类**: 解旋酶（橙色）、DNA聚合酶（绿色）、SSB蛋白（黄色）

### 4. 知识测验
- **5道测试题**: 检验对DNA复制过程的理解
- **即时反馈**: 选择答案后立即显示正确与否
- **详细解释**: 每道题都有详细的知识点解释
- **得分统计**: 完成所有题目后显示得分

### 5. 响应式设计
- 适配桌面端和移动端
- 清晰的文字显示
- 合理的色彩搭配
- 按钮文字水平垂直居中

## 安装和运行

### 环境要求
- Node.js 24.15.0 或更高版本
- npm 11.12.1 或更高版本
- Angular CLI 21.2.9

### 安装依赖

```bash
cd dna-replication-animation
npm install
```

### 启动开发服务器

```bash
npm start
```

或者使用Angular CLI命令：

```bash
ng serve
```

启动后，在浏览器中访问 `http://localhost:4200/` 即可查看应用。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist/` 目录。

### 运行测试

```bash
npm test
```

## 使用说明

1. **观看动画：
   - 点击"播放"按钮自动播放所有步骤
   - 使用"上一步"和"下一步"按钮手动控制
   - 点击顶部的步骤指示器直接跳转到特定步骤
   - 使用速度选择器调整动画播放速度

2. **查看分子结构：
   - 右侧面板包含完整的碱基图例
   - 分子组件和关键酶类的说明
   - 可展开的关键知识点折叠面板

3. **参与测验：
   - 点击"开始测验"按钮展开测验区域
   - 选择答案后立即获得即时反馈
   - 查看详细的知识点解释
   - 完成所有题目后查看得分

## 学习资源

### 关键知识点
1. **碱基互补配对原则**:
   - 腺嘌呤(A) ↔ 胸腺嘧啶(T)：2条氢键
   - 鸟嘌呤(G) ↔ 胞嘧啶(C)：3条氢键

2. **半保留复制**:
   - 每个新DNA分子包含一条旧链和一条新链
   - 确保遗传信息的准确传递

3. **5'→3'方向合成**:
   - DNA聚合酶只能沿5'→3'方向合成
   - 前导链连续合成，后随链通过冈崎片段不连续合成
