# 💰 人民币换算学习工具

一个专为小学三年级数学课程设计的互动式人民币单位换算学习工具，帮助学生直观理解元、角、分之间的换算关系。

## 📋 项目概述

本工具通过可视化的硬币操作和有趣的小游戏，让学生能够：
- 直观地添加或减少不同单位的硬币
- 体验自动进位机制（10分=1角，10角=1元）
- 理解人民币的小数表示法
- 通过小游戏巩固所学知识

## 🛠️ 技术栈

- **框架**: Angular 21+
- **语言**: TypeScript
- **样式**: Tailwind CSS 4+
- **UI 组件库**: Angular Material 21+
- **路由**: Angular Router
- **打包方式**: Angular CLI Application Builder

## 🎯 核心功能

### 1. 硬币操作区
- 展示元、角、分三种单位的硬币
- 点击 "+" 按钮添加硬币，点击 "-" 按钮减少硬币
- 自动进位功能：当分达到10个时自动转换为1角，当角达到10个时自动转换为1元
- 可视化硬币展示，让学生直观看到硬币数量变化

### 2. 金额显示
- 同时显示汉字形式（如：3元5角2分）和小数形式（如：3.52元）
- 帮助学生理解两种表示方法的对应关系
- 小数点后第一位是角，第二位是分

### 3. 换算规则提示
- 清晰展示元、角、分之间的换算关系
- 提供实用的学习小贴士
- 帮助学生记住"十进率"的换算规则

### 4. 小游戏环节
- **三级难度**：
  - 初级：仅含元（0-5元），限时60秒
  - 中级：含元、角（0-10元），限时90秒
  - 高级：含元、角、分（0-20元），限时120秒
- **游戏机制**：给出目标金额，学生通过操作硬币来匹配
- **计分系统**：答对得分，用时越短分数越高
- **暂停功能**：支持游戏暂停和继续
- **成绩统计**：游戏结束后显示详细成绩统计和评级

### 5. 使用说明
- 分步学习指南
- 人民币单位换算表
- 常见问题解答
- 学习小贴士
- 快速导航到练习和游戏

## 🚀 快速开始

### 前置要求

确保你的开发环境已安装：
- Node.js (推荐 v18 或更高版本)
- npm (通常随 Node.js 一起安装)

### 安装步骤

1. **进入项目目录**
```bash
cd rmb-converter
```

2. **安装依赖**
```bash
npm install
```

### 运行项目

**开发模式**
```bash
npm start
```

项目启动后，在浏览器中访问 `http://localhost:4200` 即可使用。

**构建生产版本**
```bash
npm run build
```

构建产物将输出到 `dist/` 目录。

**监听模式开发**
```bash
npm run watch
```

### 其他命令

| 命令 | 说明 |
|------|------|
| `npm test` | 运行单元测试 |
| `npm run build` | 构建生产版本 |
| `npm run watch` | 监听模式构建 |

## 🎨 界面设计

### 色彩方案
- **元**: 金色 (#EAB308) - 代表价值最高的单位

## 📚 教育价值

### 适合人群
- 小学三年级学生
- 需要学习人民币换算的初学者
- 希望通过互动方式学习的儿童

3. 掌握金额的两种表示方法（汉字和小数）
4. 能够进行简单的人民币计算
5. 培养数学思维和逻辑推理能力

1. **课堂引入**：先用使用说明页面介绍基本概念
2. **互动练习**：让学生在硬币操作区自由探索
3. **观察发现**：引导学生观察自动进位机制
4. **游戏巩固**：通过小游戏检验学习效果
5. **总结回顾**：使用换算表强化记忆
