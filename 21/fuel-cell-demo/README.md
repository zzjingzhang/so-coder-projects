# 酸性氢氧燃料电池演示

一个交互式的化学教学演示网页，展示酸性氢氧燃料电池的工作原理和反应过程。

## 项目简介

本项目是为高中化学教学设计的交互式演示网页，通过直观的动画展示酸性氢氧燃料电池的工作原理。学生可以观察到氢气和氧气如何在电池中反应，以及电子如何流动形成电流。

## 功能特性

- **动态燃料电池模型**：展示燃料电池的主要组成部分（正极、负极、电解质区域）
- **粒子动画效果**：
  - 氢气分子 (H₂) 的吸附和分解
  - 电子 (e⁻) 在电路中的流动
  - 氢离子 (H⁺) 在电解质中的迁移
  - 氧气分子 (O₂) 的还原过程
  - 水分子 (H₂O) 的生成过程
- **实时电流表**：显示电池工作时的电流变化
- **反应方程式标注**：清晰标注负极、正极反应及总反应式
- **交互控制**：
  - 开始/暂停按钮
  - 重置按钮，方便重新观察整个过程
- **图例说明**：详细解释图中各粒子和组件的含义
- **知识点总结**：帮助学生理解燃料电池的工作原理

## 安装和运行

### 环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0 或 yarn >= 1.22.0

### 安装依赖

```bash
cd fuel-cell-demo
npm install
```

### 开发模式

```bash
npm run dev
```

项目将在 `http://localhost:5173` 启动。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

## 使用说明

1. 打开网页后，点击页面中央的 **"开始"** 按钮启动燃料电池演示
2. 观察以下过程：
   - 氢气分子从左侧进入燃料电池，在负极分解为氢离子和电子
   - 电子通过外电路从负极流向正极，形成电流（可在电流表中观察）
   - 氢离子通过电解质从负极移向正极
   - 氧气分子从右侧进入燃料电池，在正极与氢离子和电子结合生成水
3. 点击 **"暂停"** 按钮可以暂停动画
4. 点击 **"重置"** 按钮可以重置整个演示，重新开始观察

## 化学反应方程式

- **负极反应（氧化反应）**：2H₂ - 4e⁻ → 4H⁺
- **正极反应（还原反应）**：O₂ + 4H⁺ + 4e⁻ → 2H₂O
- **总反应**：2H₂ + O₂ → 2H₂O + 电能

## 知识点

- **负极（阳极）**：氢气失去电子，被氧化生成氢离子
- **正极（阴极）**：氧气得到电子，被还原与氢离子结合生成水
- **电子流动**：从负极通过外电路流向正极，形成电流
- **离子迁移**：H⁺通过电解质从负极移向正极
- **能量转换**：化学能直接转换为电能，效率高

## 开发说明

- 主要动画逻辑在 `src/components/FuelCell.tsx` 中实现
- 使用 SVG 绘制燃料电池图形和粒子
- 使用 React 的 `useState`、`useEffect`、`useRef` 和 `useCallback` 管理状态和动画
- 动画使用 `requestAnimationFrame` 实现流畅的粒子运动
