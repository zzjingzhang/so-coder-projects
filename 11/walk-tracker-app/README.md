# 步行减肥追踪应用 (Walk Tracker App)

一款帮助用户通过步行进行减肥的应用程序，支持记录身体信息、设置减肥目标、生成个性化的每日/每周步行计划。

## 功能特性

- **身体信息录入**：记录身高、体重、年龄、性别等基础信息，并自动计算 BMI
- **减肥目标设置**：设定目标体重、每周减重目标和目标期限
- **个性化步行计划**：根据身体信息和减肥目标，智能生成每日或每周步行计划
- **计划追踪**：标记计划完成状态，追踪步数和卡路里消耗
- **数据统计**：实时统计总步数、消耗卡路里、计划完成率等数据

## 页面说明

### 1. 首页 (Home)
- 显示用户的统计数据（总步数、消耗卡路里、计划完成率）
- 展示当前身体状况和 BMI 指数
- 显示减肥目标和减重进度
- 引导用户完成设置流程

### 2. 身体信息页面 (BodyInfo)
- 录入性别、身高、体重、年龄信息
- 实时计算并显示 BMI 指数
- 提供 BMI 参考范围说明

### 3. 目标设置页面 (Goal)
- 设置目标体重
- 滑动选择每周减重目标（0.25kg - 1.5kg）
- 设定目标日期
- 提供目标分析和建议

### 4. 步行计划页面 (Plan)
- 生成每日或每周步行计划
- 显示今日计划详情（步数、时长、卡路里）
- 标签页切换查看每日计划和周计划
- 标记计划完成/未完成状态
- 统计周计划完成情况

## 智能算法

### BMI 计算
```
BMI = 体重(kg) / (身高(m))²
```

BMI 分类：
- ＜18.5: 偏瘦
- 18.5-23.9: 正常
- 24-27.9: 偏胖
- ≥28: 肥胖

### 基础步数计算
根据年龄、身高、体重智能调整基础步数：
- 年龄 < 30岁: 10000 步
- 30岁 ≤ 年龄 < 50岁: 9000 步
- 年龄 ≥ 50岁: 8000 步

额外调整：
- 体重 > 80kg: +1000 步
- 身高 > 180cm: +500 步

### 目标步数计算
根据每周减重目标计算额外需要的步数：
- 每消耗 7700 卡路里 ≈ 减重 1kg
- 每步约消耗 0.05 卡路里

```
额外步数 = (每周减重目标 * 7700 / 7) / 0.05
目标步数 = 基础步数 + 额外步数
```

## 使用流程

1. **首次使用**：
   - 打开应用，点击「开始使用 - 填写身体信息」
   - 填写身高、体重、年龄、性别
   - 设置减肥目标（目标体重、每周减重、目标日期）
   - 生成每日或每周步行计划

2. **日常使用**：
   - 查看首页统计数据
   - 在计划页面标记当日计划完成
   - 定期更新体重和目标

## 开发说明

### 状态管理
使用 React Context + useReducer 进行全局状态管理：
- `AppState`: 应用全局状态
- `AppAction`: 状态更新动作
- `AppProvider`: 状态提供者组件
- `useAppContext`: 自定义 Hook 访问状态

### 类型定义
所有数据结构都有完整的 TypeScript 类型定义：
- `BodyInfo`: 身体信息
- `Goal`: 减肥目标
- `WalkPlan`: 步行计划
- `AppState`: 应用状态
