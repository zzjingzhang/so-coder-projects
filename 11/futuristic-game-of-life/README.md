# 未来感生命游戏 (Futuristic Game of Life)

一个基于 Conway's Game of Life 的高分辨率实现，支持 GPU 加速，具有未来感的视觉效果和交互控制。

## 项目简介

本项目是一个现代化的生命游戏实现，具有以下特点：

- **GPU 加速**: 使用 WebGL 技术在 GPU 上并行计算生命游戏的下一代状态，大幅提升性能
- **高分辨率**: 支持 200x200 及更高分辨率的网格
- **未来感配色**: 采用霓虹蓝、紫色等未来感色彩，配合发光效果和脉冲动画
- **交互控制**: 提供速度控制滑块和随机化频率控制滑块
- **响应式设计**: 支持桌面和移动设备
- **双模式渲染**: 同时支持 GPU (WebGL) 和 CPU 渲染模式，自动降级

## 技术栈

- **框架**: Angular 21+

- **样式**: Tailwind CSS v4

## 核心功能

### 1. GPU 加速渲染

- 使用 WebGL 着色器在 GPU 上并行计算生命游戏规则
- 支持 200x200 = 40,000 个细胞的实时模拟
- 自动检测 WebGL 支持，不支持时自动降级到 CPU 模式

### 2. 交互控制

- **速度控制**: 滑块控制游戏速度（1-100 FPS）
- **随机化频率**: 滑块控制随机改变细胞状态的频率（0-50%）
- **开始/暂停**: 一键控制游戏运行状态
- **重置**: 重置游戏到初始状态
- **随机化**: 随机生成新的初始状态

### 3. 视觉效果

- **未来感配色**: 采用霓虹蓝 (#00f0ff)、紫色 (#9d00ff) 等赛博朋克风格色彩
- **发光效果**: 细胞具有发光边框和阴影效果
- **脉冲动画**: 细胞颜色随时间脉动变化
- **渐变背景**: 深色渐变背景增强未来感

### 4. 生命游戏规则

- **存活**: 存活细胞周围有 2-3 个存活邻居时继续存活
- **死亡**: 存活细胞周围少于 2 个（孤独）或多于 3 个（过度拥挤）邻居时死亡
- **繁殖**: 死亡细胞周围恰好有 3 个存活邻居时变为存活

## 安装与运行

### 前置要求

- Node.js 18+
- npm 9+
- Angular CLI 21+

### 安装依赖

```bash
cd futuristic-game-of-life
npm install
```

### 开发模式运行

```bash
ng serve
```

然后在浏览器中打开 `http://localhost:4200`

### 生产构建

```bash
ng build
```

构建产物将输出到 `dist/` 目录

### 运行测试

```bash
ng test
```

## 使用说明

1. **启动游戏**: 点击"开始"按钮或工具栏中的播放图标
2. **调整速度**: 使用速度滑块调整游戏运行速度（1-100）
3. **随机化**: 使用随机化频率滑块控制随机改变细胞状态的频率
4. **暂停**: 点击"暂停"按钮或工具栏中的暂停图标
5. **重置**: 点击重置按钮将游戏恢复到初始状态
6. **随机生成**: 点击随机化按钮生成新的随机初始状态

## 性能优化

### GPU 模式

- 使用 WebGL 着色器并行计算所有细胞的下一代状态
- 纹理数据在 GPU 内存中，减少数据传输开销
- 适合高分辨率网格（200x200 及以上）

### CPU 模式

- 纯 JavaScript/TypeScript 实现
- 使用双重缓冲技术优化内存使用
- 适合低分辨率网格或不支持 WebGL 的设备
