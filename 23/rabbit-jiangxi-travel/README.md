# 🐰 小兔子江西游记

一个展示江西自然风光、特色美食、非物质文化遗产和古建筑的精品专栏网站。

## 📋 项目简介

"小兔子江西游记"是一个以"小兔子"为形象的江西文化旅游展示专栏，通过长图海报、横屏视频、H5互动等多种形式，全方位展现江西的"物华天宝、人杰地灵"。

### ✨ 专栏特色

- **12期精彩内容**：覆盖江西全省11个设区市
- **三种呈现形式**：长图海报、横屏视频、H5互动作品
- **四大内容主题**：
  - 🏔️ **自然风光**（山水湖林）
  - 🍜 **特色美食**（各设区市特色美食）
  - ⭐ **非物质文化遗产**（各设区市非遗）
  - 🏛️ **古建筑**（各设区市古建筑）

## 🚀 快速开始

### 环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0

### 安装依赖

```bash
cd rabbit-jiangxi-travel
npm install
```

### 启动开发服务器

```bash
npm run dev
```

项目将在 `http://localhost:5173` 启动（端口可能会根据实际情况自动调整）。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

### 代码检查

```bash
npm run lint
```

## 🎯 设计特色

### 色彩方案
- **主色调**：深绿色 (`#2d5a3d`) - 代表江西的绿水青山
- **暖色调**：暖棕色 (`#d4a574`) - 代表江西的传统文化底蕴
- **背景色**：米白色 (`#faf8f5`) - 营造温馨舒适的阅读氛围

### 字体选择
- **标题字体**：Noto Serif SC（思源宋体）- 体现传统文化感
- **正文字体**：Noto Sans SC（思源黑体）- 保证良好的阅读性

### 交互设计
- 卡片悬停效果：上移 + 阴影加深
- 按钮悬停效果：颜色渐变
- 平滑滚动
- 响应式设计（支持移动端、平板、桌面端）

## 📝 开发说明

### 路径别名
项目已配置路径别名 `@` 指向 `src` 目录，可以更方便地导入模块：

```typescript
// 以前
import { Header } from '../../components/Header';

// 现在
import { Header } from '@/components/Header';
```

### 添加新内容
如需添加新的专栏内容，请编辑 `src/data/episodesData.ts` 文件，按照现有格式添加新的期数数据。

### 自定义样式
全局样式在 `src/index.css` 中定义，使用 Tailwind CSS 的 `@import` 指令和自定义 CSS 变量。
