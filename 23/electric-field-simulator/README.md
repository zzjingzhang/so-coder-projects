# 双电荷电场模拟器 (Electric Field Simulator)

一个交互式的双电荷电场模拟工具，帮助用户直观地理解电荷相互作用产生的电场分布。

## 项目简介

本项目是一个基于 Vue 3 的物理模拟工具，通过可视化的方式展示双电荷系统中的电场线分布。用户可以通过控制面板调整电荷的大小、极性以及两个电荷之间的距离，实时观察电场线的变化。

## 技术栈

- **前端框架**: Vue 3 (Composition API)
- **开发语言**: JavaScript
- **构建工具**: Vite
- **样式方案**: Tailwind CSS
- **UI 组件库**: Naive UI
- **路由管理**: Vue Router 5
- **可视化**: Canvas API

## 功能特性

### 核心功能

1. **电荷参数调整**
   - 调整电荷 Q1 的电量大小（1-10）
   - 调整电荷 Q1 的极性（正/负）
   - 调整电荷 Q2 的电量大小（1-10）
   - 调整电荷 Q2 的极性（正/负）

2. **间距控制**
   - 控制两个电荷之间的距离（50-300 像素）

3. **实时可视化**
   - 使用 Canvas 绘制电场线
   - 电场线上带有箭头标记指示方向
   - 参数调整后实时更新电场分布

### 视觉设计

- **正电荷**: 红色圆形 `#ff4444`
- **负电荷**: 蓝色圆形 `#4444ff`
- **电场线**: 浅蓝色半透明 `rgba(0, 100, 255, 0.4)`
- **箭头标记**: 深蓝色 `rgba(0, 100, 255, 0.9)`
- **控制面板**: 浅灰色背景 `bg-gray-100`
- **控制卡片**: 白色卡片带轻微阴影

## 物理原理

### 库仑定律

电场线的计算基于库仑定律，电场强度的叠加原理：

对于空间中任意一点 P，由多个点电荷产生的总电场是各个点电荷单独产生的电场的矢量叠加：

```
E_total = E1 + E2
```

其中每个点电荷在距离 r 处产生的电场为：

```
E = k * q / r² * (单位矢量)
```

- k: 库仑常数（本项目中取 1000 作为缩放因子）
- q: 电荷量
- r: 距离电荷的距离

### 电场线特性

- 电场线从正电荷出发，终止于负电荷
- 电场线不会相交（同一位置电场方向唯一）
- 电场线的密度表示电场强度大小
- 电场线与等势面垂直

## 安装与运行

### 环境要求

- Node.js >= 16.0.0
- npm >= 7.0.0

### 安装依赖

```bash
cd electric-field-simulator
npm install
```

### 开发模式运行

```bash
npm run dev
```

项目将在本地启动，默认访问地址为 `http://localhost:5173`

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录

### 预览生产版本

```bash
npm run preview
```

## 使用说明

1. 打开应用后，左侧为控制面板，右侧为电场可视化区域

2. **调整电荷 Q1**:
   - 使用滑动条调整电量大小（1-10）
   - 选择正电荷（+）或负电荷（-）

3. **调整电荷 Q2**:
   - 使用滑动条调整电量大小（1-10）
   - 选择正电荷（+）或负电荷（-）

4. **调整电荷间距**:
   - 使用滑动条调整两个电荷之间的距离（50-300 像素）

5. 观察右侧可视化区域：
   - 红色圆形表示正电荷
   - 蓝色圆形表示负电荷
   - 浅蓝色线条表示电场线
   - 箭头指示电场方向

## 常见场景示例

### 1. 异种电荷相互吸引（默认配置）
- Q1: 正电荷，电量 5
- Q2: 负电荷，电量 5
- 间距: 150px
- 电场线从正电荷出发，指向负电荷

### 2. 同种电荷相互排斥
- Q1: 正电荷，电量 5
- Q2: 正电荷，电量 5
- 间距: 150px
- 电场线从两个正电荷向外发散，中间区域相互排斥

### 3. 不等量电荷
- Q1: 正电荷，电量 10
- Q2: 负电荷，电量 3
- 间距: 200px
- 观察电场线如何被大电荷主导
