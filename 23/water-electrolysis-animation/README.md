# 水电解动画教学演示

一个交互式的水电解过程动画网页，用于化学教育展示。

## 项目简介

本项目是一个教育性的水电解动画网页，通过5个步骤的交互式动画，清晰展示水分解为氢气和氧气的完整化学过程。项目采用现代化的前端技术栈，具有响应式设计，适合在各种设备上进行教学演示。

## 技术栈

- **框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **样式方案**: Tailwind CSS
- **UI组件库**: PrimeVue
- **路由**: Vue Router
- **语言**: JavaScript

## 功能特性

### 动画步骤
1. **初始状态**: 展示电解槽、电极和水的初始状态
2. **接通电源**: 展示电路连接和电流开始流动
3. **电子流动**: 可视化电子从阴极流向水分子，从水分子流向阳极的过程
4. **分子分解**: 展示水分子分解为氢离子和氢氧根离子
5. **气体产生**: 展示氢气和氧气气泡的产生，突出2:1的体积比

### 交互功能
- 上一步、下一步按钮控制动画进度
- 重置按钮回到初始状态
- 步骤指示器快速跳转任意步骤
- 进度条显示当前位置
- 思考问题折叠/展开功能

### 教育内容
- 完整的化学方程式展示
- 阴极和阳极的半反应方程式
- 氢气和氧气2:1体积比的视觉演示
- 3个深入思考问题及详细答案

## 安装与运行

### 环境要求
- Node.js (推荐 v16 或更高版本)
- npm (随 Node.js 一起安装)

### 安装依赖

```bash
cd water-electrolysis-animation
npm install
```

### 开发模式

```bash
npm run dev
```

项目将在本地启动，默认地址为 `http://localhost:5173` (端口可能会根据实际情况变化)

### 构建生产版本

```bash
npm run build
```

构建完成后，文件将输出到 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

## 视觉设计亮点

- **蓝色调主题**: 符合水环境的视觉预期
- **渐变效果**: 使用 Tailwind CSS 的渐变功能创建美观的视觉层次
- **响应式布局**: 最大宽度800px，在手机和桌面设备上均有良好表现
- **SVG动画**: 原生SVG动画实现流畅的气泡上升、电子流动等效果
- **步骤指示器**: 根据进度变换颜色（已完成:绿色, 当前:蓝色, 未完成:灰色）
- **色彩编码**:
  - 阴极: 蓝色系
  - 阳极: 红色系
  - 氢气: 蓝色边框气泡
  - 氧气: 红色边框气泡
  - 氧原子: 红色圆形
  - 氢原子: 蓝色圆形

## 使用说明

1. 打开网页后，您将看到电解水的初始状态
2. 点击**下一步**按钮逐步查看电解过程
3. 观察每个步骤的动画效果和文字说明
4. 点击步骤指示器上的数字可直接跳转到对应步骤
5. 点击**重置**按钮可随时回到初始状态
6. 阅读页面下方的化学方程式加深理解
7. 点击思考问题的展开按钮查看详细答案

## 化学方程式参考

**总反应方程式:**
```
2H₂O → 2H₂↑ + O₂↑
```

**阴极反应 (还原反应):**
```
4H₂O + 4e⁻ → 2H₂↑ + 4OH⁻
```

**阳极反应 (氧化反应):**
```
2H₂O → O₂↑ + 4H⁺ + 4e⁻
```

## 开发建议

- 如需修改动画效果，请编辑 `src/views/WaterElectrolysis.vue` 中的 SVG 部分
- 如需添加新步骤，请修改 `steps` 数组和对应的 `v-if` 条件
- 样式调整主要在 `style.css` 和组件内的 `<style scoped>` 中进行
- 动画时长和效果可通过修改 SVG 的 `<animate>` 标签参数调整
