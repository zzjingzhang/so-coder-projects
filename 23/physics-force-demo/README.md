# 物理力学教学演示 - 二力平衡与运动状态

这是一个交互式的物理力学教学演示工具，用于展示二力平衡与运动状态的关系。

## 项目概述

本项目通过可视化的方式演示了物理力学中的二力平衡原理。当物体受到的拉力（F）和重力（G）大小相等、方向相反时，物体处于平衡状态。此时物体可以保持静止，也可以做匀速直线运动。

## 技术栈

- **框架**: Vue 3

## 功能特性

### 左侧控制面板
- **运动控制按钮**：
  - 向上移动按钮（绿色，带上箭头符号）
  - 向下移动按钮（红色，带下箭头符号）
  - 停止按钮（灰色，带方块符号）
  
- **速度控制滑块**：
  - 范围：1-20
  - 初始值：10
  - 显示格式：1.0 m/s（实际值为滑块值除以10）
  
- **重力控制滑块**：
  - 范围：5-30N
  - 初始值：10N
  
- **状态显示**：实时显示当前运动状态和二力平衡原理说明

### 右侧演示区域
- **白色背景容器**，带黑色边框
- **顶部圆形电动机**：带动绳索和重物运动
- **绳索**：连接电动机和重物
- **圆形重物**：蓝色圆形，带有"重物"标签
- **力的示意图**：
  - 向上的红色拉力 F，带箭头和数值标签
  - 向下的蓝色重力 G，带箭头和数值标签
  - 力的箭头长度随重力大小动态变化
- **底部说明文字**：解释二力平衡原理

### 交互功能
- 点击向上/向下按钮时，重物沿绳索匀速直线运动
- 点击停止按钮时，重物立即停止运动
- 调整速度滑块时，实时改变重物的运动速度
- 调整重力滑块时，力的箭头长度和标签数值同步更新
- 响应式设计，适配不同屏幕尺寸

## 安装与运行

### 环境要求
- Node.js 16.0 或更高版本
- npm 或 yarn 包管理器

### 安装依赖

```bash
cd physics-force-demo
npm install
```

### 开发模式

```bash
npm run dev
```

项目启动后，在浏览器中访问显示的本地地址（通常是 `http://localhost:5173`）即可查看演示页面。

### 构建生产版本

```bash
npm run build
```

构建完成后，生成的文件将位于 `dist` 目录中。

### 预览生产版本

```bash
npm run preview
```

预览构建后的生产版本。

## 使用说明

1. **启动项目**：按照上述步骤安装依赖并启动开发服务器
2. **控制运动**：使用左侧控制面板的三个按钮控制重物的运动状态
3. **调整速度**：拖动速度滑块改变重物的运动速度（范围：0.1-2.0 m/s）
4. **调整重力**：拖动重力滑块改变力的大小（范围：5-30N），观察力的箭头长度变化
5. **观察现象**：注意在二力平衡状态下，物体既可以静止，也可以做匀速直线运动

## 物理原理说明

本演示展示了**二力平衡**的物理原理：

- **二力平衡条件**：作用在同一物体上的两个力大小相等、方向相反，并且作用在同一直线上
- **平衡状态**：物体受到平衡力作用时，总保持静止状态或匀速直线运动状态
- **演示中的平衡力**：
  - 拉力 F（红色，向上）：电动机通过绳索对重物的拉力
  - 重力 G（蓝色，向下）：地球对重物的吸引力
  - F = G 时，二力平衡

这个演示帮助学生理解：**物体的运动状态改变需要力的作用，但物体的运动不需要力来维持**。在平衡力作用下，物体可以保持匀速直线运动。

## 开发说明

### 核心文件说明

- `src/views/PhysicsForceDemo.vue`：主要的演示页面组件，包含所有交互逻辑和UI展示
- `src/router/index.ts`：路由配置，当前只有一个演示页面
- `src/main.ts`：应用入口，配置Vue、Vuetify和Vue Router
- `src/style.css`：全局样式，引入Tailwind CSS

### 响应式设计

项目使用Tailwind CSS实现响应式布局：
- 移动端：垂直布局（控制面板在上，演示区在下）
- 平板/桌面端：水平布局（控制面板在左，演示区在右）

### 动画实现

使用 `requestAnimationFrame` API 实现平滑的动画效果，确保重物运动流畅且性能良好。
