# 法拉第电磁感应定律演示

一个用于高中物理教学的交互式网页动画，展示法拉第电磁感应定律中感应电流产生的条件。

## 项目简介

本项目是一个直观的物理教学演示工具，通过动画形式展示了金属导体在不同运动状态下与磁感线的相互作用，帮助学生理解感应电流产生的核心条件。

### 主要功能

- **交互式演示区域**：包含磁场（N极、S极）、可移动的金属棒和电流表
- **三种运动状态**：
  - 静止状态
  - 沿磁感线运动
  - 切割磁感线运动
- **实时反馈**：通过电流表指针变化直观展示感应电流的产生
- **详细解释**：底部包含实验结论和物理原理解释

## 技术栈

- **框架**: React 19
- **语言**: TypeScript
- **样式**: Tailwind CSS 4
- **UI 组件库**: Chakra UI 3

## 安装与运行

### 前置要求

- Node.js 18+ 版本
- npm 或 pnpm 包管理器

### 安装依赖

```bash
# 进入项目目录
cd faraday-law-demo

# 安装依赖
npm install

# 或者使用 pnpm
pnpm install
```

### 开发模式运行

```bash
npm run dev
```

应用将在 `http://localhost:5173` 启动。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

## 使用说明

1. 打开网页后，您将看到一个磁场区域，包含 N 极（红色）和 S 极（蓝色），以及中间的磁感线。
2. 中间有一根可移动的金属棒，右下角是电流表。
3. 点击不同的控制按钮：
   - **静止**：金属棒保持静止，电流表指针指向零位
   - **沿磁感线运动**：金属棒沿上下方向运动（与磁感线平行），不产生感应电流
   - **切割磁感线运动**：金属棒沿左右方向运动（切割磁感线），产生感应电流，电流表指针摆动
4. 观察不同状态下电流表指针的变化，对比理解感应电流产生的条件。
5. 查看页面底部的实验结论和详细解释，加深对法拉第电磁感应定律的理解。

## 核心物理原理

**法拉第电磁感应定律**：当穿过闭合电路的磁通量发生变化时，电路中就会产生感应电动势，从而产生感应电流。

**感应电流产生的条件**：
1. 电路必须是闭合的
2. 穿过电路的磁通量必须发生变化

**切割磁感线运动**：当导体在磁场中运动，且运动方向与磁感线不平行时，导体就会切割磁感线，导致穿过闭合电路的磁通量发生变化，从而产生感应电流。
