# 抽奖机项目

一个基于 React 的互动抽奖机应用。

## 功能特性

- 抽奖机主体带有轮盘，轮盘上分布多个问号，问号以抽奖机主体中心为中心环绕排列，且问号四周有圆环围绕
- 轮盘通过阴影滚动效果实现随机动画
- 抽奖机右侧设有按钮，点击按钮后开始随机过程
- 主体左侧有一个洞，随机过程结束后会从洞中弹出一个纸团，纸团边缘有发光效果
- 点击纸团，纸团会缓慢展开并显示在画面中央，展开后中央展示一个数字
- 数字规则：
  - 范围在 1 到 46 之间
  - 排除 39 和 45 两个数字
  - 第一次弹出的数字固定为 19
  - 后续弹出的数字为上述范围内的纯随机数
- 数字需在纸团展开后的正中间清晰显示
- 纸团展开后再次点击，纸团会消失，随后可重复抽奖流程

## 技术栈

- **React**: 前端框架
- **Vite**: 打包工具
- **Tailwind CSS**: 样式框架
- **Chakra UI**: UI 组件库
- **React Router**: 路由管理

## 安装和启动

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

开发服务器启动后，在浏览器中访问 `http://localhost:5173` 即可查看应用。

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 使用说明

1. 点击右侧的圆形按钮开始抽奖
2. 轮盘会开始旋转并展示随机动画效果
3. 抽奖结束后，左侧会弹出一个带有发光效果的纸团
4. 点击纸团，纸团会缓慢展开并显示在画面中央
5. 展开后的纸团中央会显示一个数字
6. 再次点击纸团或背景，纸团会消失，然后可以进行下一次抽奖

## 注意事项

- 第一次抽奖的结果固定为 19
- 后续抽奖结果为 1-46 之间的随机数，但不会出现 39 和 45
- 每次抽奖后需要关闭纸团才能进行下一次抽奖
