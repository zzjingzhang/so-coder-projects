# Notion 风格块编辑器

一个基于 Vue 3、TypeScript、Tailwind CSS 和 PrimeVue 构建的 Notion 风格块编辑器。

## 功能特性

- **多种块类型**：支持标题（1-3级）、段落、待办事项、嵌入、表格、分割线等多种块类型
- **拖放排序**：通过拖拽手柄轻松重新排序块
- **块类型转换**：快速将文本块转换为标题或待办事项
- **本地存储**：自动保存内容到浏览器本地存储
- **响应式设计**：适配不同屏幕尺寸

## 安装与运行

### 前置要求

- Node.js >= 18.0.0
- npm >= 9.0.0

### 安装依赖

```bash
cd notion-block-editor
npm install
```

### 开发模式

```bash
npm run dev
```

应用将在 `http://localhost:5173` 启动。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

## 使用说明

### 添加块

1. 点击块左侧的 **+ 按钮** 打开块类型菜单
2. 选择所需的块类型（文本、标题、待办事项、嵌入、表格、分割线）
3. 或者在空段落块中输入 `/` 也可以打开菜单

### 编辑内容

- 直接点击文本区域即可开始编辑
- 标题块支持 1-3 级大小
- 待办事项块可以勾选/取消勾选

### 重新排序块

1. 将鼠标悬停在块上，左侧会显示操作按钮
2. 点击并按住 **六点图标**（拖拽手柄）
3. 拖动块到目标位置后释放

### 转换块类型

1. 将鼠标悬停在块上
2. 点击右侧的 **转换图标**（左右箭头）
3. 选择目标类型（文本/标题/待办事项）

> 注意：嵌入块、表格块和分割线块不能转换为其他类型

### 删除块

1. 将鼠标悬停在块上
2. 点击右侧的 **删除图标**（垃圾桶）

> 注意：至少会保留一个块

### 嵌入内容

嵌入块支持以下类型：

- **网页 (iframe)**：嵌入任意网页
- **YouTube 视频**：粘贴 YouTube 链接自动转换为嵌入视频
- **图片**：嵌入网络图片

### 表格操作

表格块支持：
- 编辑单元格内容
- 添加/删除行
- 添加/删除列

## 块类型说明

| 类型 | 说明 | 快捷键/操作 |
|------|------|-------------|
| 文本 | 普通段落文本 | Enter 键创建新段落 |
| 一级标题 | 大号标题 | 可转换自文本块 |
| 二级标题 | 中号标题 | 可转换自文本块 |
| 三级标题 | 小号标题 | 可转换自文本块 |
| 待办事项 | 带复选框的任务列表 | 点击复选框切换状态 |
| 嵌入 | 嵌入网页、视频或图片 | 选择类型并输入 URL |
| 表格 | 可编辑表格 | 支持添加/删除行列 |
| 分割线 | 水平分割线 | 用于分隔内容区域 |

## 数据持久化

编辑器内容会自动保存到浏览器的 `localStorage` 中，键名为 `notion-blocks`。刷新页面后内容会自动恢复。

手动保存：点击顶部工具栏的 **保存** 按钮会触发手动保存并显示确认提示。

清空内容：点击顶部工具栏的 **清空** 按钮会重置编辑器到初始状态。
