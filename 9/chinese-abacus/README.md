# 中国传统算盘模拟器

一个精美的传统中国算盘（上二下五型）网页模拟器，体验古老的计算智慧。

## 项目简介

本项目是一个基于 React 构建的传统中国算盘模拟器，实现了上二下五型算盘的完整功能。算盘是中国古代劳动人民发明创造的一种简便的计算工具，被誉为"世界上最古老的计算机"。

## 功能特性

- **12档算盘**：从个位到千亿位，满足大数计算需求
- **上二下五结构**：每档包含2个上珠（每个代表5）和5个下珠（每个代表1）
- **实时计算**：点击珠子后实时计算并显示总数值，包含千位分隔符
- **复位功能**：一键将所有珠子重置到初始位置
- **精美动画**：珠子移动有平滑过渡动画，点击有细腻的交互反馈
- **响应式设计**：适配不同屏幕尺寸，在各类设备上均可正常使用
- **使用说明**：详细的操作指南，帮助用户快速上手

## 技术栈

- **前端框架**：React 19
- **构建工具**：Vite 8
- **样式方案**：Tailwind CSS 4
- **动画库**：Framer Motion 12
- **路由**：React Router 7
- **语言**：JavaScript (JSX)

## 核心组件说明

### App.jsx

主应用组件，包含以下核心功能：

1. **状态管理**：使用 `useState` 管理12档算盘的珠子状态
2. **数值计算**：`calculateValue` 函数实时计算当前激活珠子的总数值
3. **事件处理**：
   - `handleUpperClick`：处理上珠点击事件
   - `handleLowerClick`：处理下珠点击事件
   - `handleReset`：复位所有珠子到初始状态

### 子组件

- **Bead**：珠子组件，处理单个珠子的显示和交互
- **AbacusColumn**：算盘列组件，管理每一档的上下珠
- **InfoCard**：使用说明卡片组件
- **DecorativeElement**：装饰元素组件

## 使用说明

### 基本操作

1. **上珠操作**：点击横梁上方的上珠，珠子会向下移动并变为红色。每个上珠代表数值5。
2. **下珠操作**：点击横梁下方的下珠，珠子会向上移动并变为蓝色。每个下珠代表数值1。
3. **取消激活**：再次点击已激活的珠子，可以取消其激活状态。
4. **复位归零**：点击红色的「复位归零」按钮，将所有珠子重置到初始位置。

### 数值计算规则

- 每档的数值 =（激活的上珠数 × 5）+（激活的下珠数 × 1）
- 总数值 = 各档数值 × 对应位值之和
- 位值从右到左依次为：1、10、100、1000、...、100000000000（千亿）

### 示例

- 表示数字 7：在个位档激活1个上珠（5）和2个下珠（1+1），总计 5+1+1=7
- 表示数字 123：
  - 个位档：3个下珠（3）
  - 十位档：2个下珠（2×10=20）
  - 百位档：1个下珠 + 0个上珠（1×100=100）
  - 总计：100+20+3=123

## 开发指南

### 添加新功能

1. 在 `App.jsx` 中添加新的状态或逻辑
2. 如需创建新组件，建议在 `src` 目录下创建 `components` 文件夹
3. 样式优先使用 Tailwind CSS 工具类，复杂样式可在 `index.css` 中添加

### 自定义配色

在 `src/index.css` 的 `@theme` 区块中修改颜色变量：

```css
@theme {
  --color-gold: #d4af37;       /* 金色 */
  --color-red: #c41e3a;         /* 红色 */
  --color-blue: #1e90ff;        /* 蓝色 */
  --color-wood-light: #deb887;  /* 浅木色 */
  --color-wood-dark: #8b4513;   /* 深木色 */
  /* ... 其他颜色 */
}
```
