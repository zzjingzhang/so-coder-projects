# 体育联赛积分榜

一个功能完善的体育联赛积分榜应用，展示球队排名、积分、净胜球、近期状态、晋级/降级区，并支持动画排名变化效果。

## 功能特性

- **球队排名**: 实时显示球队排名，支持动画排名变化
- **详细数据**: 展示场次、胜平负、进球、失球、净胜球、积分等详细数据
- **近期状态**: 显示球队最近5场比赛的胜负平状态
- **晋级/降级区**: 不同颜色标识欧冠区、欧联杯区、欧协联区和降级区
- **模拟比赛**: 点击"模拟一轮比赛"按钮，观察排名变化动画

## 技术栈

- **框架**: Vue 3
- **语言**: JavaScript

- **UI组件库**: Vuetify 3

## 安装与运行

### 前置要求

- Node.js >= 16.0.0

```bash

```

### 开发模式运行

```bash
npm run dev
```

应用将在 `http://localhost:5173` 运行

### 构建生产版本

```bash
npm run build
```


## 使用说明

1. **查看积分榜**: 打开应用即可看到英超联赛2023-2024赛季的积分榜
2. **理解颜色标识**:
   - 绿色左侧边框: 欧冠区 (前4名)
   - 蓝色左侧边框: 欧联杯区 (第5名)
   - 橙色左侧边框: 欧协联区 (第6名)
   - 红色左侧边框: 降级区 (最后3名)
3. **模拟比赛**: 点击"模拟一轮比赛"按钮，系统将随机生成比赛结果，并显示排名变化动画
4. **排名变化**: 
   - 绿色向上箭头: 排名上升
   - 红色向下箭头: 排名下降
   - 短横线: 排名不变

## 核心功能实现

### 排名算法

球队排名按照以下优先级排序：
1. 积分 (高 → 低)
2. 净胜球 (多 → 少)
3. 进球数 (多 → 少)

### 动画效果

- 排名上升/下降时的数字动画
- 表格行的颜色标识
- 状态芯片的颜色区分

### 数据结构

每个球队对象包含以下属性：
- `id`: 球队ID
- `name`: 球队名称
- `logo`: 球队Logo URL
- `played`: 已赛场次
- `wins`: 胜场
- `draws`: 平场
- `losses`: 负场
- `goalsFor`: 进球数
- `goalsAgainst`: 失球数
- `goalDifference`: 净胜球
- `points`: 积分
- `form`: 近期状态 (最近5场)
- `currentRank`: 当前排名
- `previousRank`: 上轮排名
