# 硫酸铜溶液与铁的氧化还原反应交互展示

一个用于化学教学的交互式网页，展示硫酸铜溶液与铁发生氧化还原反应的过程，包括电子移动方向、电子移动速度与硫酸铜溶液浓度的探究。

## 功能特性

### 1. 反应控制
- **开始反应**: 启动氧化还原反应动画
- **停止反应**: 暂停当前反应
- **重置反应**: 重置所有反应状态

### 2. 参数调节
- **硫酸铜溶液浓度**: 可调节范围 0.1 - 2.0 mol/L，浓度越高反应速率越快
- **电子移动速度**: 可调节范围 1x - 10x，控制电子转移动画的速度

### 3. 实时状态展示
- 反应状态（进行中/未开始）
- 铁的消耗进度条
- 铜的生成进度条
- 电子转移数量统计

### 4. 动画展示
- 硫酸铜溶液颜色随浓度变化（浓度越高，蓝色越深）
- 铜离子（Cu²⁺）在溶液中的分布
- 电子（e⁻）从铁（Fe）向铜离子（Cu²⁺）的移动动画
- 铁钉表面生成铜单质的过程

### 5. 反应原理说明
- **氧化反应**: Fe - 2e⁻ → Fe²⁺ （铁失去电子，被氧化）
- **还原反应**: Cu²⁺ + 2e⁻ → Cu （铜离子得到电子，被还原）
- **总反应**: Fe + CuSO₄ → FeSO₄ + Cu
- **浓度影响**: 解释浓度对反应速率的影响原理

## 安装与运行

### 环境要求
- Node.js 版本 >= 16.0.0
- npm 或 yarn 或 pnpm

### 安装依赖

```bash
cd redox-reaction-demo
npm install
```

### 开发模式运行

```bash
npm run dev
```

项目启动后，浏览器访问 http://localhost:5173 即可查看。

### 构建生产版本

```bash
npm run build
```

构建完成后，产物位于 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

## 使用说明

1. **调节参数**: 在开始反应前，可先调节硫酸铜溶液的浓度
2. **开始反应**: 点击"开始反应"按钮，观察反应过程
3. **调节速度**: 反应进行中可调节电子移动速度
4. **查看原理**: 点击下方标签页查看详细的反应原理说明
5. **重置实验**: 点击"重置"按钮可重新开始实验

## 化学知识点

### 氧化还原反应基本概念
- **氧化剂**: 得到电子，化合价降低，发生还原反应（Cu²⁺ → Cu）
- **还原剂**: 失去电子，化合价升高，发生氧化反应（Fe → Fe²⁺）
- **电子转移**: 电子从还原剂转移到氧化剂

### 金属活动性顺序
K > Ca > Na > Mg > Al > Zn > **Fe** > Ni > Sn > Pb > H > **Cu** > Hg > Ag > Pt > Au

排在前面的金属可以将排在后面的金属从其盐溶液中置换出来。

### 浓度对反应速率的影响
根据碰撞理论：
- 反应物浓度越大，单位体积内活化分子数越多
- 有效碰撞频率越高
- 反应速率越快
