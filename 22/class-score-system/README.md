# 班级积分系统

一个专门为小学班级设计的可爱卡通风格积分管理系统，支持小组积分和学生个人积分双模式管理。

## ✨ 功能特点

### 👥 小组积分页面
- **4个小组**：每个小组有独特的颜色和可爱的卡通图案（🐰🐻🐸🦊）
- **积分调节**：加分/减分按钮，数值可自由调节（1-5分，默认1分）
- **小组名称**：可自由输入自定义小组名称
- **排序功能**：实时显示积分排名前三的小组
- **皇冠标识**：积分第一名的小组旁显示皇冠标识👑
- **天空背景**：漂浮云朵与渐变天空背景，风格可爱卡通

### 👨‍🎓 学生个人积分页面
- **38名学生**：预设38名学生，可自由分配到4个小组
- **积分转换**：组内学生共积攒10分即可为小组积分增加1分
- **进度显示**：各小组积分转换进度条可视化
- **分组筛选**：点击小组卡片可筛选显示对应小组的学生
- **学生管理**：支持修改学生姓名、单个学生加减分

## 🚀 快速开始

### 环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0

### 安装依赖

```bash
cd class-score-system
npm install
```

### 启动开发服务器

```bash
npm run dev
```

启动后在浏览器访问 `http://localhost:5173`

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 📝 使用说明

### 小组积分操作

1. **调整积分值**：点击页面顶部的 `+` 或 `-` 按钮设置每次加减的分数（1-5分）
2. **修改小组名称**：直接点击小组名称处即可编辑
3. **加减积分**：点击红色 `-` 按钮减分，点击绿色 `+` 按钮加分
4. **查看排名**：点击 "显示排行榜" 按钮查看积分前三的小组
5. **重置数据**：点击顶部导航栏的 "重置" 按钮可清空所有积分

### 学生积分操作

1. **调整积分值**：点击页面顶部的 `+` 或 `-` 按钮设置每次加减的分数
2. **修改学生姓名**：直接点击学生姓名处即可编辑
3. **筛选小组**：点击顶部小组进度卡片可筛选显示对应小组的学生
4. **积分转换**：当小组内学生积分累计达到10分时，系统会自动为该小组增加1分，并弹出提示消息

## 💾 数据存储

- 所有数据（小组积分、学生积分、小组名称、学生名称等）都保存在浏览器的 localStorage 中
- 刷新页面后数据不会丢失
- 如需清空数据，请使用页面顶部的 "重置" 按钮

## ⚙️ 自定义配置

### 修改默认小组

如需修改默认小组配置，请编辑 `src/types/index.ts` 中的 `DEFAULT_TEAMS` 变量：

```typescript
export const DEFAULT_TEAMS: Team[] = [
  { id: 1, name: '第一组', score: 0, color: 'pink', emoji: '🐰' },
  // ...
]
```

### 修改学生数量

如需修改学生数量，请编辑 `src/types/index.ts` 中的 `generateDefaultStudents` 函数。
