# 🎮 推箱子游戏

一个适合家庭亲子互动的经典推箱子网页游戏。

## 📖 项目简介

这是一个为退休小学老师和孙子一起玩而设计的推箱子游戏。游戏具有明亮友好的界面、多个难度关卡、步数统计、重新开始功能，以及自定义关卡创建功能。

## 🛠️ 技术栈

- **框架**: Vue 3 (Composition API)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **UI 组件库**: Vuetify 3
- **路由**: Vue Router 4
- **打包工具**: Vite

## 🚀 快速开始

### 环境要求

- Node.js >= 16.0.0
- npm >= 7.0.0

### 安装依赖

```bash
cd push-box-game
npm install
```

### 启动开发服务器

```bash
npm run dev
```

启动后，在浏览器中访问 `http://localhost:5173` 即可开始游戏。

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 🎯 游戏功能

### 1. 多个难度关卡

- **简单关卡**: 2关，适合小朋友入门
- **中等关卡**: 2关，逐步增加难度
- **困难关卡**: 1关，挑战逻辑思维

### 2. 游戏控制

- **屏幕按钮**: 大尺寸方向按钮，适合触屏操作
- **键盘控制**: 支持方向键或 WASD 键
- **步数统计**: 实时显示移动步数

### 3. 重新开始功能

- 随时可以点击"重新开始"按钮重置当前关卡

### 4. 自定义关卡编辑器

- 可以创建自己的关卡
- 支持放置墙壁、玩家、箱子、目标
- 可以保存和加载自定义关卡
- 关卡数据保存在浏览器本地存储中

## 🎨 界面特点

- **明亮友好**: 使用绿色、橙色等温暖色调
- **大按钮**: 适合小朋友操作
- **Emoji 图标**: 直观可爱的游戏元素（😊、📦、🧱）
- **响应式设计**: 支持电脑和平板设备

## 📖 游戏规则

1. 控制角色（😊）移动
2. 把所有箱子（📦）推到目标位置（绿色圆点）上
3. 箱子只能推，不能拉
4. 尝试用最少的步数完成每一关

## 💡 操作提示

- **电脑用户**: 可以用方向键或 WASD 键控制
- **平板用户**: 点击屏幕上的方向按钮
- **自定义关卡**: 在编辑器中设计好关卡后，点击"开始游戏"即可游玩

## 📝 开发说明

### 添加新关卡

编辑 `src/data/levels.ts` 文件，按照现有格式添加新关卡数据。

### 修改主题颜色

在 `src/main.ts` 中修改 Vuetify 的主题配置。
