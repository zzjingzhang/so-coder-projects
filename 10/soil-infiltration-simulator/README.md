# 🌧️ 土壤下渗实验模拟

一个交互式的土壤下渗实验模拟器，用于展示不同类型土壤（黏土、黄土、砂土）的水分渗透能力差异。

## 📋 项目简介

本项目通过可视化的方式模拟了雨水降落到不同类型土壤上的下渗过程，实时显示下渗量和速度，并使用图表记录数据，帮助用户理解土壤颗粒大小与水分渗透能力的关系。

### 主要功能

- **三种土壤类型模拟**：黏土、黄土、砂土，每种土壤具有不同的颗粒大小和渗透特性
- **实时降雨模拟**：可视化的雨滴效果，可调节降雨强度
- **下渗过程展示**：直观展示水分在土壤中的渗透深度
- **收集过程显示**：显示渗透到土壤底部并被收集的水量
- **实时数据监测**：显示下渗速度、已下渗量、已收集量等实时数据
- **数据图表记录**：使用 ECharts 实时绘制三种土壤的下渗量对比曲线
- **实验控制**：支持开始、暂停、重置实验，调节降雨强度

## 📖 使用说明

### 控制面板

1. **实验时间**：显示实验已运行的时间（分:秒格式）
2. **降雨强度**：显示当前设定的降雨强度（mm/h）
3. **开始实验**：启动模拟，开始降雨和下渗过程
4. **暂停**：暂停当前实验，保持当前状态
5. **重置**：重置所有数据，回到初始状态
6. **降雨强度调节**：通过滑块调节降雨强度（范围：10-100 mm/h）

### 实验区域

页面展示三个并列的土壤模拟器，分别对应：
- **黏土**：颗粒极细，渗透性最差
- **黄土**：颗粒中等，渗透性适中
- **砂土**：颗粒粗大，渗透性最好

每个模拟器下方显示实时数据：
- **已下渗**：已经渗透进入土壤的水量（ml）
- **已收集**：渗透到土壤底部并被收集的水量（ml）
- **下渗速度**：当前的水分渗透速度（ml/s）

### 数据图表

图表实时记录三种土壤的下渗量随时间的变化，方便进行对比分析。

### 土壤类型说明

页面底部提供了三种土壤类型的详细说明，包括：
- 颗粒大小范围
- 典型下渗速度
- 特性描述
- 实际应用场景

## 🔬 实验原理

### 土壤渗透能力的决定因素

土壤的渗透能力主要取决于以下因素：

1. **颗粒大小**：颗粒越大，颗粒之间的孔隙越大，水分越容易通过
2. **孔隙度**：土壤中孔隙体积占总体积的比例
3. **颗粒排列**：土壤颗粒的排列方式影响孔隙的连通性

### 三种土壤的特性对比

| 土壤类型 | 颗粒大小 | 下渗速度 | 渗透性 | 保水性 |
|----------|----------|----------|--------|--------|
| 黏土 | 0.002mm 以下 | ~1.5 ml/s | 最差 | 最好 |
| 黄土 | 0.002-0.05mm | ~8 ml/s | 适中 | 一般 |
| 砂土 | 0.05-2mm | ~25 ml/s | 最好 | 最差 |

### 模拟算法

本模拟使用简化的物理模型：

1. **降雨计算**：根据设定的降雨强度计算单位时间内的降雨量
2. **下渗计算**：根据土壤类型的渗透系数计算最大可能下渗量
3. **水量平衡**：
   - 地表水 = 降雨量 - 下渗量
   - 土壤含水量 += 下渗量
   - 当土壤饱和时，多余水分开始流出并被收集

## 🎨 组件说明

### SoilSimulator.vue

土壤模拟器核心组件，负责：
- 渲染降雨动画
- 模拟水分在土壤中的渗透过程
- 计算和更新下渗数据
- 可视化展示地表水、土壤水、收集水

**Props：**
- `soilType` (String, required): 土壤类型 ('clay' | 'loess' | 'sand')
- `isRunning` (Boolean): 是否正在运行模拟
- `rainfallIntensity` (Number): 降雨强度（mm/h）

**Events：**
- `update:data`: 实时返回下渗数据

### DataChart.vue

数据图表组件，使用 ECharts 实现：
- 实时绘制三种土壤的下渗量曲线
- 支持图例切换
- 响应式布局

**Props：**
- `chartData` (Object): 图表数据对象

### ExperimentView.vue

实验主页面，整合所有组件：
- 提供实验控制面板
- 布局三个土壤模拟器
- 管理实验数据
- 显示土壤类型说明

## 📝 开发说明

### 土壤类型配置

土壤类型配置位于 `src/utils/soilTypes.js`，可以根据需要调整参数：

```javascript
export const soilTypes = {
  clay: {
    name: '黏土',
    particleSize: '0.002mm 以下',
    particleSizeMm: 0.002,
    infiltrationRate: 1.5,      // 下渗速度 (ml/s)
    maxInfiltration: 50,         // 最大下渗量 (ml)
    porosity: 0.3,               // 孔隙度
    color: '#8B4513',            // 土壤颜色
    lightColor: '#CD853F',       // 浅色（用于颗粒）
    description: '黏土颗粒极细...'
  },
  // ... 其他土壤类型
}
```

### 添加新的土壤类型

1. 在 `soilTypes.js` 中添加新的土壤配置
2. 在 `ExperimentView.vue` 中添加对应的模拟器实例
3. 在 `DataChart.vue` 中更新图表配置

## 🐛 常见问题

**Q: 为什么图表没有显示数据？**

A: 请确保：
1. 实验已经开始运行（点击"开始实验"按钮）
2. 已经运行了至少 1 秒钟（数据每秒记录一次）
3. 检查浏览器控制台是否有错误信息

**Q: 如何调整模拟速度？**

A: 目前模拟速度是固定的。如果需要调整，可以修改 `SoilSimulator.vue` 中的动画逻辑，或调整 `soilTypes.js` 中的 `infiltrationRate` 参数。

**Q: 支持移动端吗？**

A: 是的，项目使用 Tailwind CSS 的响应式设计，支持各种屏幕尺寸。在移动端，三个土壤模拟器会垂直排列。
