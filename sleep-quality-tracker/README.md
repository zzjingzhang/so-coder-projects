# 睡眠质量追踪器

一款功能完整的睡眠质量追踪应用，帮助您记录、分析和改善睡眠质量。

## 功能特性

### 📊 核心功能
- **睡眠数据记录** - 记录上床时间、起床时间、睡眠环境因素等
- **拖拽调整时间段** - 通过拖拽滑块直观调整睡眠时间段
- **睡眠阶段分析** - 可视化展示深睡、浅睡、REM、清醒等睡眠阶段
- **质量评分系统** - 根据睡眠时长、睡眠阶段和环境因素计算睡眠质量评分
- **睡眠历史日历** - 日历视图查看历史睡眠记录，点击日期查看详情

### 🎯 目标与成就
- **睡眠目标设置** - 设定目标睡眠时长、理想睡眠时间段
- **成就系统** - 完成目标解锁成就，激励持续追踪
- **连续记录追踪** - 查看连续记录天数和优质睡眠连续天数

### 💡 智能建议
- **个性化睡眠建议** - 根据睡眠数据提供针对性的改善建议
- **环境因素记录** - 记录温度、湿度、噪音、光线、咖啡因摄入、运动、压力水平等

### ⌨️ 键盘快捷键
- `Ctrl + S` - 快速导航到记录页面
- `Tab` - 在表单字段间切换
- `Enter` - 保存睡眠记录

### 💾 数据管理
- **本地存储** - 使用 localStorage 持久化存储数据
- **数据导出** - 导出睡眠数据为 JSON 文件
- **数据导入** - 从 JSON 文件导入睡眠数据
- **示例数据** - 一键加载30条示例数据体验完整功能

## 技术栈

- **框架**: Vue 3 (Composition API)
- **状态管理**: Pinia
- **路由**: Vue Router 4
- **UI 组件库**: PrimeVue
- **样式**: Tailwind CSS
- **图表**: Chart.js
- **构建工具**: Vite
- **语言**: JavaScript

## 页面说明

- 显示昨夜睡眠概览（时长、评分）

### 3. 历史记录 (HistoryView)
- 月历视图展示睡眠记录
- 点击日期查看详细睡眠数据
- 睡眠阶段饼图
- 环境因素详情
- 编辑/删除记录功能

### 4. 目标与成就 (GoalsView)
- 睡眠目标设置（时长、时间、温度、湿度等）
- 成就系统展示（6个成就）
- 目标达成追踪统计

### 5. 设置 (SettingsView)
- 数据统计概览
- 数据导入/导出功能
- 加载示例数据
- 清除所有数据（带确认）
- 键盘快捷键说明
- 关于信息

## 安装与运行

- Node.js >= 16.0.0

```bash
npm install
```

```bash
npm run dev
```


构建产物将输出到 `dist` 目录

```bash

```

## 使用指南

### 首次使用
1. 打开应用后，导航到「设置」页面
2. 点击「加载示例数据」体验完整功能
3. 或者直接前往「记录睡眠」开始记录

### 记录睡眠
1. 选择日期（默认为今天）
2. 调整上床和起床时间：
   - 直接在时间输入框中选择
   - 或者在时间轴上点击/拖拽滑块
3. 填写环境因素（可选但推荐）
4. 添加备注（可选）
5. 点击「保存记录」或按 `Enter` 键

### 查看历史
1. 导航到「历史记录」页面
2. 在日历中点击有记录的日期
3. 右侧面板显示详细信息，包括睡眠阶段图表

### 设置目标
1. 导航到「目标与成就」页面
2. 调整目标睡眠时长和时间段
3. 设置理想的温度和湿度
4. 点击「保存目标设置」

## 睡眠质量评分说明

评分基于以下因素计算（满分100分）：

| 因素 | 条件 | 加分/减分 |
|------|------|----------|
| 睡眠时长 | 7-9小时 | +30 |
| 睡眠时长 | 6-7小时 或 9-10小时 | +15 |
| 睡眠时长 | <5小时 或 >11小时 | -10 |
| 深睡时长 | >=90分钟 | +10 |
| REM时长 | >=90分钟 | +10 |
| 环境安静 | 噪音为安静 | +5 |
| 环境黑暗 | 光线为黑暗 | +5 |
| 温度适宜 | 16-20°C | +5 |
| 湿度适宜 | 40-60% | +5 |

## 成就系统

| 成就名称 | 解锁条件 |
|---------|---------|
| 第一周 | 连续记录睡眠一周 |
| 周优质睡眠 | 连续7天获得良好睡眠质量（70分以上） |
| 月优质睡眠 | 连续30天获得良好睡眠质量 |
| 完美睡眠 | 获得一次95分以上的完美睡眠 |
| 早起达人 | 7天在5-7点之间起床 |
| 环境专家 | 记录10次完整的睡眠环境数据 |

## 数据存储

所有数据存储在浏览器的 localStorage 中，包括：
- `sleep_records` - 睡眠记录
- `sleep_goals` - 睡眠目标设置
- `sleep_achievements` - 已解锁的成就

注意：清除浏览器缓存会删除所有本地数据，建议定期导出备份。
