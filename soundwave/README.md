# SoundWave - 音频播放器应用

一个类似 SoundCloud 的现代化音频播放器应用，采用 Vue 3 + TypeScript + Vite 技术栈构建，支持响应式布局。

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **TypeScript** - JavaScript 的超集，提供类型安全
- **Vite** - 下一代前端构建工具
- **Vue Router 4** - Vue.js 官方路由管理器
- **CSS3** - 现代 CSS 特性（Flexbox、Grid、响应式设计）

## 页面结构

### 1. 首页 (Home)
- **Hero 区域**：应用介绍和探索入口
- **精选播放列表**：展示推荐的播放列表
- **热门歌曲**：展示当前热门歌曲
- **新发布**：展示最新音乐
- **音乐分类**：按音乐类型浏览

### 2. 搜索页 (Search)
- **搜索框**：支持搜索歌曲、艺术家、专辑、流派
- **分类浏览**：按音乐分类快速筛选
- **全部歌曲**：展示所有可用歌曲
- **全部播放列表**：展示所有播放列表
- **搜索结果**：实时显示搜索匹配的歌曲和播放列表

### 3. 播放列表详情页 (Playlist)
- **播放列表信息**：封面、标题、描述、创建者
- **播放控制**：全部播放、随机播放
- **歌曲列表**：详细的歌曲信息列表
- **播放状态指示**：当前播放歌曲高亮显示

### 4. 个人资料页 (Profile)
- **用户信息**：头像、用户名、粉丝/关注统计
- **喜欢的歌曲**：展示用户喜欢的歌曲
- **我的播放列表**：展示用户创建的播放列表
- **最近播放**：展示用户最近播放的歌曲

### 核心组件
- **NavBar**：顶部导航栏，支持移动端菜单
- **BottomPlayer**：底部音频播放器，全局固定
- **TrackCard**：歌曲卡片组件
- **PlaylistCard**：播放列表卡片组件

## 功能特性

### 音频播放器功能
- ✅ 播放/暂停控制
- ✅ 上一首/下一首切换
- ✅ 进度条拖拽跳转
- ✅ 音量调节
- ✅ 静音切换
- ✅ 随机播放
- ✅ 单曲循环
- ✅ 播放时间显示（当前时间/总时长）
- ✅ 播放状态同步

### UI/UX 特性
- ✅ 深色主题设计（类似 SoundCloud 风格）
- ✅ 流畅的动画和过渡效果
- ✅ 悬停交互反馈
- ✅ 卡片式布局
- ✅ 响应式设计（支持移动端、平板、桌面端）
- ✅ 移动端优化的导航和播放器

### 响应式布局
- **桌面端**：完整的多列布局，所有功能可用
- **平板端**：自适应列数，优化导航体验
- **移动端**：单列布局，精简播放器控制，底部导航

## 如何启动项目

### 前置要求
- Node.js >= 16.0.0
- npm (Node 包管理器)

### 安装依赖

```bash
cd soundwave
npm install
```

### 开发模式

```bash
npm run dev
```

项目将在 `http://localhost:5173` 启动。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### 预览生产构建

```bash
npm run preview
```

## 设计亮点

### 视觉设计
- **深色主题**：采用深色背景配合橙色主色调，营造音乐应用氛围
- **渐变效果**：标题栏和分类卡片使用渐变色彩
- **卡片阴影**：悬停时的阴影效果提升层次感
- **图标系统**：统一的 SVG 图标风格

### 交互设计
- **平滑过渡**：所有状态变化都有流畅的过渡动画
- **悬停反馈**：卡片和按钮都有悬停状态
- **播放指示器**：当前播放歌曲有视觉指示器
- **加载状态**：图片加载有平滑过渡

### 移动端适配
- **响应式网格**：根据屏幕宽度自动调整列数
- **精简界面**：移动端隐藏次要功能
- **触摸友好**：按钮和可点击区域足够大
- **导航优化**：移动端支持汉堡菜单

### Vue Router 路由管理
- 基于文件的路由

### 现代化 CSS
- CSS 变量定义主题
- Flexbox 和 Grid 布局
- 媒体查询响应式设计
- CSS 过渡和动画
